## Getting Started

1. Clone the repository from https://github.com/mephisto83/bailey-code
2. Install dependencies using 'npm install' or 'yarn install' to install all 23 dependencies including React, TypeScript, and ProjectStorm diagrams
3. Run 'npm start' or 'yarn start' to start the development server on http://localhost:3000
4. For testing, run 'npm test' or 'yarn test' to launch the Jest test runner with React Testing Library
5. To build for production, run 'npm run build' or 'yarn build' which will create optimized build files in the build directory and execute the custom fixreport.js post-processing script
6. The application will be available with full AdminLTE styling, interactive diagram capabilities, and all CSS modules properly typed with TypeScript declarations
