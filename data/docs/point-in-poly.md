# point-in-poly

![Language](https://img.shields.io/badge/language-TypeScript-blue)

## Overview

Point-in-Poly is a TypeScript library that implements geometric algorithms for determining whether a point lies inside a polygon. The library uses the ray casting algorithm (also known as the crossing number algorithm) to solve the point-in-polygon problem, which is a fundamental computational geometry challenge. The implementation handles various edge cases including collinear points and boundary intersections, making it robust for real-world geometric calculations in applications like GIS, computer graphics, and spatial analysis.

## Architecture

The library follows a modular TypeScript architecture with four core components: Point class (src/point.ts) that represents 2D coordinates, GFG class (src/cfg.ts) containing the main geometric algorithms, a test module (src/test.ts) for validation, and an entry point (src/index.ts) that exports the functionality. The GFG class implements the ray casting algorithm by casting a ray from the test point to infinity and counting intersections with polygon edges. Key methods include orientation() for determining point relationships, doIntersect() for line segment intersection detection, onSegment() for collinearity checks, and isInside() as the main point-in-polygon function. The build process uses TypeScript compilation followed by a custom pack-up.js script for version management.

## Key Features

- Ray casting algorithm implementation for point-in-polygon detection
- Robust handling of edge cases including collinear points and boundary intersections
- Support for arbitrary polygon shapes with any number of vertices
- Orientation detection for clockwise, counterclockwise, and collinear point relationships
- Line segment intersection calculation using cross product mathematics
- Comprehensive test suite with multiple polygon shapes (rectangles, triangles) and various point positions
- TypeScript type safety for geometric Point objects with x,y coordinates
- Automated version bumping through custom pack-up.js build script

## Getting Started

1. Clone the repository from https://github.com/mephisto83/point-in-poly.git
2. Install dependencies with 'npm install' (only TypeScript 4.1.2 is required as devDependency)
3. Build the project using 'npm run build' which compiles TypeScript and runs pack-up.js for versioning
4. Run tests with 'npm run run' to execute the test cases in src/test.ts
5. For development, use 'npm run watch' to enable TypeScript watch mode for automatic compilation
6. Import the library in your project using the default export from src/index.ts which provides access to the CFG class containing the geometric algorithms
