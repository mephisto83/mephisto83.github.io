## Overview

Style Extractor is a TypeScript-based browser tool designed to simplify the process of duplicating and extracting styled components from existing webpages. The tool analyzes DOM elements and their computed CSS styles, then generates reusable component code for both React and vanilla HTML implementations. It operates entirely in the browser console, making it a practical utility for developers who need to reverse-engineer or replicate existing UI components without manually inspecting and copying CSS properties.

## Architecture

The application follows a single-file architecture with the main logic contained in src/index.ts, which compiles to browser-executable JavaScript. The core system consists of several interconnected functions: getAllStyles() extracts all computed CSS properties from DOM elements, getDefaultProperty() determines default browser styles to filter out non-custom properties, buildStylesForElement() processes and formats styles, and crawlElement() recursively traverses the DOM tree to build component hierarchies. The src/pr.ts file contains pre-processed style data with device-specific responsive breakpoints (iPhone 5, Laptop, Desktop) and hover states, suggesting the tool can handle complex responsive designs and interactive states.

## Key Features

- Extracts computed CSS styles from any DOM element using CSS selectors
- Filters out default browser styles to only capture custom styling
- Generates React-compatible component code with camelCase style properties
- Produces vanilla HTML with inline stylesheets as an alternative output
- Recursively processes child elements to create complete component trees
- Handles text nodes with customizable text wrapper functions
- Deduplicates identical styles by creating reusable CSS classes
- Extracts and normalizes color values (RGB, hex) into reusable variables
- Supports responsive design patterns with device-specific breakpoints
- Preserves element attributes while excluding style and class attributes

## Getting Started

1. Clone the repository from GitHub. 2. Run 'npm install' to install the TypeScript dependency. 3. Install TypeScript globally if not already available: 'npm install -g typescript'. 4. Compile the TypeScript source code by running 'tsc' in the project root directory. 5. Open your browser's developer console and copy-paste the compiled JavaScript output from the generated files. 6. Execute the tool using 'style_extractor({selector: 'your-css-selector', reactMode: true})' for React output or 'style_extractor({selector: 'your-css-selector'})' for inline CSS output, replacing 'your-css-selector' with the target element's CSS selector. 7. The tool will output the extracted component code to the browser console.
