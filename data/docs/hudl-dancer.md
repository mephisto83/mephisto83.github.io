# hudl-dancer

![Language](https://img.shields.io/badge/language-TypeScript-blue)

## Overview

hudl-dancer is a Chrome Extension built with Manifest V3 that provides video analysis capabilities, specifically designed to work with video players that use the 'theo-player-wrapper' class structure. The extension enables users to create rectangular selections on video content and extract image frames from videos, suggesting functionality for sports video analysis or annotation workflows. Based on the React Chrome Extension boilerplate, it includes content scripts, background processes, popup interfaces, and developer tools integration for comprehensive video interaction capabilities.

## Architecture

The extension follows a modular Chrome Extension architecture with separate pages for different contexts: Background scripts (src/pages/Background/index.js) handle extension lifecycle, Content scripts (src/pages/Content/) inject UI components and video interaction logic into web pages, Popup (src/pages/Popup/) provides the main user interface, Options (src/pages/Options/) handles settings, Panel (src/pages/Panel/) integrates with Chrome DevTools, and Newtab (src/pages/Newtab/) customizes the new tab page. Key components include hooks.ts for video position tracking and rectangle selection, util.ts for video frame extraction, and various React components like RectangleCreator.tsx, BoxList.tsx, Panel.tsx, and Typeahead.tsx that handle the interactive video analysis features.

## Key Features

- Video position tracking using IntersectionObserver API to monitor theo-player-wrapper video containers
- Interactive rectangle selection tool with mouse-based drawing capabilities for marking video regions
- Video frame extraction functionality that captures still images from HTML5 video elements and converts them to downloadable files
- Chrome DevTools integration for debugging and development workflows
- Hot reload development environment with automatic extension reloading
- TypeScript support with intelligent Chrome API code completion
- Multi-context extension architecture supporting content scripts, popup, options, background, and new tab pages
- Responsive UI components including typeahead search and dynamic box lists

## Getting Started

1. Ensure Node.js version >= 18 is installed. 2. Clone the repository from https://github.com/mephisto83/hudl-dancer.git. 3. Update package.json fields: change 'name' from 'hudl-dancer', update 'description', and modify 'repository.url' to your fork. 4. Edit src/manifest.json to customize the extension name and permissions. 5. Run 'npm install' to install all dependencies including React, TypeScript, Webpack, and Chrome extension development tools. 6. Run 'npm start' to launch the development server with hot reload on localhost:3000. 7. Open Chrome and navigate to chrome://extensions/, enable Developer mode, click 'Load unpacked extension', and select the generated 'build' folder. 8. For production builds, run 'NODE_ENV=production npm run build' to generate optimized extension files and a distributable ZIP package in the zip/ directory.
