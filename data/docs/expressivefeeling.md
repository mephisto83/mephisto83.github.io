# expressivefeeling

![Language](https://img.shields.io/badge/language-TypeScript-blue)

## Overview

ExpressiveFeeling is a comprehensive emotional wellness and art therapy platform that combines mobile and web applications with Firebase backend services. The platform enables users to create emotional associations through visual art, audio recordings, and interactive experiences. It provides guided sessions where users progress through various emotional prompts, creating digital collages and recordings that help them process and express their feelings. The system includes multi-platform support with a React Native mobile app (icanfeel), a React web application, and Firebase Functions for backend processing.

## Architecture

The system follows a multi-tier architecture with separate mobile (mobile/icanfeel), web (web/), and backend (funcs/) applications. The mobile app uses Expo and React Native with Firebase integration for real-time data synchronization. The web application is built with React, Material-UI, and Redux for state management. Firebase Functions serve as the backend API layer, handling user authentication, data persistence, and business logic through auto-generated controllers in funcs/functions/src/controllers/. The architecture includes Firebase Firestore for data storage, Firebase Storage for media assets, and Firebase Hosting for web deployment. A processing layer (process/) handles Google Cloud services for speech-to-text and storage operations.

## Key Features

- Interactive emotional assessment sessions with step-by-step prompts defined in constants/Prompts
- Digital collage creation with draggable images, scaling, and rotation using StaticImage components
- Audio recording and playback functionality through react-audio-voice-recorder
- Multi-platform synchronization between mobile and web applications
- Real-time canvas collaboration with fabric.js integration for drawing and image manipulation
- Customizable backgrounds and image assets stored in funcs/functions/public/image/
- Organizational multi-tenancy with role-based access control through organization agents
- Automated deployment pipeline with build and publish scripts for web and mobile platforms
- Speech-to-text processing integration with Google Cloud Speech API
- Comprehensive user session management with progress tracking through emotional association steps

## Getting Started

1. Install dependencies: Run `npm install` in root, `cd web && npm install`, `cd mobile/icanfeel && npm install`, and `cd funcs/functions && npm install`. 2. Configure Firebase: Set up Firebase project and replace configuration in funcs/functions/expressivefeeling-firebase-adminsdk-*.json. 3. Set up Google Cloud: Configure Google Cloud Speech API credentials in process/ directory. 4. Mobile setup: Install Expo CLI globally with `npm install -g @expo/cli`, then run `expo start` in mobile/icanfeel directory. 5. Web development: Run `npm start` in web/ directory to start React development server. 6. Firebase Functions: Deploy with `cd funcs && npm run deploy` after building TypeScript with `npm run build`. 7. Build and deploy web: Use `npm run web` from root to build web app and deploy to Firebase Hosting. 8. Mobile build: Run `npm run build` in mobile/icanfeel for production builds using EAS Build.
