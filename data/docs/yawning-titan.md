# yawning-titan

![Language](https://img.shields.io/badge/language-CSS-blue)

## Overview

Yawning Titan is a Chrome extension that extracts HTML elements and their associated CSS styles from existing webpages and automatically converts them into reusable React components. The tool analyzes the DOM structure, collects all relevant CSS rules from stylesheets, and generates TypeScript React components with properly formatted JSX and SCSS files. It provides a complete workflow from web element selection to component generation, making it valuable for developers who need to rapidly prototype or migrate existing web designs into React applications.

## Architecture

The project follows a three-tier architecture with a Chrome extension frontend, a local Express.js server middleware, and a TypeScript build pipeline. The chrome_extension/ directory contains the browser extension with popup.js handling user interaction, devtools.js intercepting CSS network requests, and index.js (compiled from web/src/index.ts) performing the core DOM analysis and CSS extraction. The local/ directory provides an Express server (app.js) that proxies CSS requests to bypass CORS restrictions and saves generated components to the filesystem. The web/ directory contains TypeScript source code that gets compiled and injected into the extension, with build.js handling the compilation pipeline from TypeScript to JavaScript.

## Key Features

- DOM element selection and HTML extraction with configurable depth traversal
- Comprehensive CSS rule collection from external stylesheets, inline styles, and network requests
- Automatic conversion of HTML attributes to JSX-compatible format (class to className, tabindex to tabIndex, etc.)
- React component generation with TypeScript interfaces and SCSS styling
- Local file system integration to automatically save generated components
- Chrome DevTools integration for CSS network request interception
- Configurable component naming and CSS selector targeting
- Support for both top-level wrapper components and deep nested component extraction

## Getting Started

1. Install dependencies for the local server by running 'npm install' in the local/ directory. 2. Start the local Express server on port 16552 by running 'node app.js' from the local/ directory. 3. Update the component_folder path in local/app.js to point to your desired output directory for generated components. 4. Build the TypeScript source by running 'npm run build' in the web/ directory, which compiles web/src/index.ts and copies the output to chrome_extension/index.js. 5. Load the Chrome extension by opening Chrome's extension management page, enabling Developer mode, and selecting 'Load unpacked' to point to the chrome_extension/ directory. 6. Navigate to any webpage, open the extension popup, enter a component name and CSS selector, then click the generation button to extract and save React components to your specified directory.
