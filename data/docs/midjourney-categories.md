# midjourney-categories

![Language](https://img.shields.io/badge/language-TypeScript-blue)

## Overview

The midjourney-categories project is a React-based web application designed to categorize and manage AI-generated images from Midjourney. The system appears to be built for analyzing and organizing a large collection of generated artwork, specifically focused on letter-based designs created in various colors and artistic styles. The project contains thousands of generated images (968MB repository size) with systematic naming conventions that encode metadata about the username, letter, color, and artist style used in the generation prompts.

## Architecture

The project follows a typical React application structure with two main components: a frontend React app in the 'categorize' directory and a Node.js tooling system in the 'tools' directory. The React application uses Redux for state management and includes comprehensive UI libraries for data visualization and interaction. The tools directory contains TypeScript utilities for processing the large image collection, automatically generating TypeScript modules that export image source paths. The naming convention of images follows a structured pattern: 'username_letter_spelled_out_in_color_drawn_by_artist_uuid_coordinates.png', which is parsed by utility functions to extract categorical metadata for the frontend application.

## Key Features

- Image categorization system with automated metadata extraction from filenames using regex parsing in util.ts
- Comprehensive React UI with drag-and-drop functionality via react-beautiful-dnd for organizing categories
- Data visualization capabilities through react-apexcharts for analyzing image collections
- Grid-based layout system using react-grid-layout for flexible image organization
- Automated tooling system that scans the public/images directory and generates TypeScript modules with image source paths
- Redux-based state management for handling large datasets of categorized images
- Responsive design with device detection and perfect scrollbar components
- Export functionality with react-to-print for generating reports of categorized images

## Getting Started

1. Clone the repository and navigate to the categorize directory. 2. Install React app dependencies with 'npm install' in the categorize folder. 3. Navigate to the tools directory and install tooling dependencies with 'npm install'. 4. Run 'npm run process_images' in the tools directory to generate the image_sources.ts file that indexes all PNG files in the public/images directory. 5. Return to the categorize directory and start the development server with 'npm start' to launch the application on http://localhost:3000. 6. The application will load with access to the categorized image collection and provide interfaces for sorting and analyzing the Midjourney-generated artwork based on the extracted metadata from filenames.
