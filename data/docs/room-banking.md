# room-banking

> a banking app for the class room

![Language](https://img.shields.io/badge/language-TypeScript-blue)

## Overview

Room Banking is a comprehensive educational banking simulation platform designed for classroom environments. It provides students with hands-on experience in financial management, banking operations, and economic systems through a gamified virtual city environment. The application combines traditional banking features with immersive city-building mechanics, allowing students to manage banks, purchase land, operate businesses, and participate in economic activities within a controlled educational setting.

## Architecture

The project follows a Firebase-based serverless architecture with a React frontend and TypeScript backend. The core structure consists of a React web application (./web) and Firebase Cloud Functions (./funcs) handling backend operations. The data layer uses Firestore with auto-generated CRUD controllers for entities like banks, customers, transactions, businesses, and city assets. The system implements a sophisticated model-driven architecture with auto-generated services in ./funcs/functions/src/controllers/auto/ and business logic in ./funcs/functions/src/functions/. Key architectural patterns include domain models with property definitions, denormalized data structures for performance, and event-driven updates through On_*_Update functions.

## Key Features

- Virtual banking system with account management and transactions
- City simulation with purchasable land parcels (CityLand model) and placeable assets (CityAsset model)
- Multi-entity classroom management supporting districts, schools, professors, and students
- Auction system for trading virtual assets and business opportunities
- Business planning and virtual business operations with digital asset management
- Educational course system with payment integration and progress tracking
- Real-time data synchronization through Firestore with automatic model updates
- Comprehensive user role management including bank owners, customers, directors, and room agents
- Stripe integration for handling real monetary transactions in educational contexts

## Getting Started

1. Install Node.js 16 and yarn package manager. 2. Clone the repository and install root dependencies with 'yarn install'. 3. Navigate to ./web directory and run 'yarn install' for frontend dependencies. 4. Navigate to ./funcs/functions and run 'npm install' for Firebase function dependencies. 5. Set up Firebase project and configure authentication credentials (reference: room-banking-dev-firebase-adminsdk-pgy44-41fc89dca4.json). 6. Configure Firestore database rules using ./funcs/firestore.rules and indexes from ./funcs/firestore.indexes.json. 7. For development: run 'npm run serve-all' from ./funcs to start Firebase emulators. 8. For web development: run 'yarn start' from ./web directory to start React development server. 9. For production deployment: use 'npm run web' to build and deploy the complete application, or 'npm run deploy' for Firebase functions only. 10. Configure CORS settings using ./cors.json for cross-origin requests.
