## Overview

PresentationBlendResources is a Blender automation toolkit designed for creating animated presentations and videos from image sequences. The project leverages Blender's Python API (bpy) to programmatically generate video content by combining image sequences with audio tracks through Blender's video sequence editor. This appears to be a specialized tool for automating the creation of presentation videos, likely used for converting static content into dynamic animated presentations.

## Architecture

The architecture consists of two main Python scripts that interface with Blender's API: anim_video_editor.py handles the core video compilation workflow by creating video sequences from image directories and synchronizing them with audio files, while render.py manages rendering operations with configurable frame ranges and camera settings. The system uses Blender's sequence editor to composite image sequences with audio tracks, supporting H264 video output with MP3 audio encoding. Multiple .blend files (armature_test.blend, Arms.blend, mat.blend, objects.blend, untitled.blend) serve as Blender project templates or assets, with corresponding .blend1 backup files indicating iterative development workflows.

## Getting Started

1. Install Blender with Python API support
2. Clone the repository and ensure all .blend files and Python scripts are in the same directory
3. Prepare input directories: create image sequence directory (referenced by relPath argument) and audio directory (referenced by relAudioPath argument)
4. Create an 'output' directory in the project root for rendered videos
5. Execute anim_video_editor.py via Blender's command line: `blender --background --python anim_video_editor.py -- [relPath] [frameCount] [relAudioPath] [outputPrefix]`
6. For custom rendering, use render.py: `blender [blendFile] --background --python render.py -- [startFrame] [endFrame] [cameraName]`
7. Ensure proper file permissions for the output directory and verify FFmpeg codecs are available in your Blender installation
