# torry-me

![Language](https://img.shields.io/badge/language-Python-blue)

## Overview

torry-me is a Python-based torrent downloader application that automates the process of downloading torrents from RSS feeds. The tool monitors RSS feeds containing torrent links, automatically downloads the .torrent files, and manages the downloading process using the libtorrent library. It serves as an automated solution for batch downloading content from torrent RSS feeds, eliminating the need for manual intervention in the torrent acquisition and download process.

## Architecture

The application follows a simple single-file architecture in combined_downloader.py with three main functional components: RSS feed parsing using feedparser, torrent file management through HTTP requests, and torrent downloading via libtorrent session management. The workflow processes RSS feeds to extract torrent URLs, downloads .torrent files to a local directory structure, creates a single libtorrent session to manage all torrents concurrently, and provides real-time progress monitoring. The architecture separates torrent files into a dedicated subdirectory while downloading actual content to the specified download directory.

## Tech Stack

- Python 3
- feedparser 6.0.10
- requests 2.31.0
- python-libtorrent
- libtorrent BitTorrent library

## Key Features

- RSS feed parsing to automatically discover torrent links from feed entries
- Automatic .torrent file downloading and local storage management
- Concurrent torrent downloading using a single libtorrent session
- Real-time progress monitoring with download/upload speeds and peer count
- Support for multiple torrent MIME types including application/x-bittorrent and application/octet-stream
- Automatic directory structure creation for organized file management
- Command-line interface accepting RSS feed URL and download directory parameters
- Error handling for network requests and torrent file processing

## Getting Started

1. Ensure Python 3 is installed on your system. 2. Install required dependencies by running 'pip install -r requirements.txt' which will install feedparser==6.0.10, requests==2.31.0, and python-libtorrent. 3. Note that python-libtorrent may require additional system-level dependencies depending on your operating system. 4. Run the application using 'python combined_downloader.py <rss_feed_url> <download_directory>' where rss_feed_url is the URL of the RSS feed containing torrent links and download_directory is the local path where torrents will be downloaded. 5. The application will create a 'torrent_files' subdirectory within the specified download directory to store .torrent files. 6. Monitor the console output for download progress, speeds, and completion status.
