# redquick-ai-extension

> redquick ai extension vscode extension

![Language](https://img.shields.io/badge/language-TypeScript-blue)

## Overview

RedQuick AI Extension is a Visual Studio Code extension that leverages AI to help developers build Firebase web applications through an intelligent code generation interface. The extension integrates with the Groq AI service to provide contextual assistance in setting up multi-platform projects including web, Electron, and React Native applications. It features a conversational AI panel that guides users through project initialization, workspace setup, and dependency management with automated npm/npx command execution.

## Architecture

The extension follows a modular architecture with a main extension entry point (extension.ts) that registers commands and activates the GroqCodegenPanel webview. The core intelligence is handled by an invocation engine (invocation.ts) that processes different types of user requests through a state machine pattern using the Invocations enum. The system maintains mental context about project preferences and uses structured prompts to communicate with the Groq LLM. Utility functions (utils.ts) handle file system operations, workspace management, and command execution. Documentation generation capabilities provide interface validation and automated documentation creation.

## Getting Started

1. Clone the repository and navigate to the redquick-ai directory. 2. Run 'yarn install' to install dependencies including groq-sdk and development tools. 3. Compile the extension using 'yarn run compile' which runs type checking, linting, and esbuild bundling. 4. Open the project in VS Code and press F5 to launch a new Extension Development Host window. 5. In the new window, open the Command Palette (Ctrl+Shift+P) and run 'Open Redquick AI Panel' command. 6. Obtain a Groq API key and enter it in the panel interface. 7. Use the conversational interface to specify which types of apps to create (web, electron, react-native). 8. The extension will automatically create workspace folders, initialize package.json files, and install required dependencies based on your selections.
