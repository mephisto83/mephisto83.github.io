## Overview

Veronica-lessons is a JavaScript educational repository designed as a collection of programming exercises and examples focused on fundamental JavaScript concepts. The project serves as a hands-on learning resource covering core programming principles including variable scoping, function assignment, flow control structures, and basic arithmetic operations. This appears to be a tutorial or lesson-based repository aimed at teaching JavaScript fundamentals through practical coding exercises and demonstrations.

## Architecture

The repository follows a simple flat file structure with individual JavaScript files (assignment.js, fcassignment.js, flowcontrol.js, scoping.js through scoping5.js, variables.js, and store.js) each focusing on specific JavaScript concepts. Each file can be executed independently using Node.js and demonstrates different programming patterns. The scoping files (scoping.js through scoping5.js) progressively demonstrate variable scoping concepts with var and let keywords, while flowcontrol.js implements conditional logic and loops. The assignment files contain exercise templates for students to complete basic mathematical operations and object manipulation.

- Progressive JavaScript scoping demonstrations showing differences between var and let keywords across multiple execution contexts
- Flow control examples implementing conditional statements (if/else) and iterative structures (for loops)

## Getting Started

1. Clone the repository from https://github.com/mephisto83/veronica-lessons.git. 2. Navigate to the project directory and run 'npm install' to install the command-line-args dependency. 3. Execute individual lesson files using Node.js, for example: 'node scoping2.js' to see scoping demonstrations, 'node flowcontrol.js' to run flow control examples, or 'node assignment.js' to work on the assignment exercises. 4. Complete the incomplete functions in assignment.js and fcassignment.js by implementing the requested mathematical operations and cost calculation logic. 5. Test your implementations by running the files and verifying the console output matches expected results.
