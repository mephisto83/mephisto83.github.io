# compliment

![Language](https://img.shields.io/badge/language-JavaScript-blue)

## Overview

Compliment is a React Native mobile application designed to share positive affirmations and compliments with contacts. The app provides users with a curated collection of over 200 compliments that can be randomly displayed and shared through various communication channels. Built as a cross-platform solution supporting both iOS and Android, the application aims to spread positivity by making it easy to send encouraging messages to friends and family members.

## Architecture

The application follows a Redux-Flux architecture pattern with React Native Router Flux for navigation. The main entry points (index.android.js and index.ios.js) bootstrap the app component from src/app.js, which configures the Redux store with middleware and sets up the router with scene-based navigation. The architecture includes a centralized state management system through Redux, with actions defined in src/actions/uiActions.js handling compliment loading, contact access, and UI state updates. The UI layer consists of React Native components using NativeBase for consistent styling, with the main interface in src/home.js providing tap-to-advance functionality for browsing compliments. A side drawer navigation component provides additional app functionality and settings access.

## Key Features

- Random compliment generation from a curated database of 200+ positive affirmations stored in src/compliments.js
- Tap-to-advance interface allowing users to cycle through compliments by touching the screen
- Native contact integration using react-native-contacts to access device address book
- Cross-platform support for both iOS and Android with platform-specific build configurations
- Redux state management for consistent app state across components
- Side drawer navigation menu for additional app functionality
- Communication integration through react-native-communications for sharing compliments
- Responsive design adapting to different screen dimensions
- Persistent state management with AsyncStorage integration

## Getting Started

1. Clone the repository and navigate to the compliments directory. 2. Install dependencies using 'npm install' to install React Native 0.44.0, NativeBase, Redux, and other required packages from package.json. 3. For Android development, ensure Android SDK is installed and configure the android/local.properties file with SDK path, then run 'react-native run-android' from the compliments directory. 4. For iOS development, open ios/compliments.xcodeproj in Xcode, configure signing certificates, and build the project, or run 'react-native run-ios' from command line. 5. Ensure proper permissions are configured for contact access in AndroidManifest.xml and Info.plist files. 6. The app will launch with the home screen displaying random compliments that can be navigated by tapping the screen, with contact integration requiring device permissions for full functionality.
