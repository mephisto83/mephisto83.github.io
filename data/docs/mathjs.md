# mathjs

![Language](https://img.shields.io/badge/language-JavaScript-blue)

## Overview

This is a comprehensive JavaScript mathematics library providing advanced mathematical computation capabilities including symbolic expression evaluation, linear algebra operations, and Fast Fourier Transform implementations. The library features a sophisticated expression parser and evaluator that can handle symbolic mathematics including calculus operations like integration and differentiation, trigonometric functions, and complex mathematical expressions. The project appears to be designed as a modular mathematical toolkit with both basic mathematical operations and advanced features like quaternion mathematics, matrix operations, and signal processing capabilities through FFT.

## Architecture

The project follows a modular ES6 class-based architecture with two main directory structures: `math/` and `_math/`. The core architecture centers around the Expression class (`math/expression.js`) which serves as the foundation for symbolic mathematics. The Evaluator class (`math/expressions/evaluator.js`) handles expression evaluation with recursive preprocessing and type-based evaluation dispatch. The Factor class (`math/expressions/factor.js`) provides factorization utilities and numerical analysis. Supporting modules include specialized mathematical components like Hamilton quaternions (`math/hamilton.js`), FFT implementations (`math/fft.js`), and various vector/matrix operations. The ExpressionMatch class (`math/expressionmatch.js`) implements pattern matching for symbolic integration. The project uses Babel for ES6 transpilation as evidenced by the babel configuration in package.json.

## Key Features

- Symbolic expression evaluation with support for variables, constants, and complex mathematical operations
- Comprehensive calculus support including integration and differentiation through ExpressionMatch pattern matching
- Full trigonometric and hyperbolic function support (sin, cos, tan, sec, csc, cot, sinh, cosh, tanh, sech, csch, coth)
- Fast Fourier Transform implementation with butterfly algorithms for signal processing
- Quaternion mathematics through Hamilton class with multiplication and addition operations
- Matrix operations and 3D mathematical computations via Matrix and Matrix3d classes
- Advanced factorization and numerical analysis through the Factor utility class
- Vector mathematics with support for 3D operations and J3DI vector implementations
- Logarithmic and exponential function evaluation
- Mathematical expression parsing and symbolic manipulation
- Summation and power operations with recursive evaluation

## Getting Started

1. Clone the repository from https://github.com/mephisto83/mathjs
2. Install dependencies by running `npm install` which will install babel-cli, babel-core, and babel-preset-es2015
3. Transpile the ES6 source code using `npm run trans` which executes babel to convert src to dest directory
4. Import the required mathematical modules using ES6 import syntax, for example: `import Expression from './math/expression'`, `import Evaluator from './math/expressions/evaluator'`, `import Hamilton from './math/hamilton'`
5. The library can be used for symbolic mathematics by creating Expression objects and using the Evaluator class for computation
6. Note: No test suite is currently implemented as indicated by the package.json test script
