## Overview

Discord Dancer is a Chrome extension (Manifest V3) built with React 18 and Webpack 5 that appears to provide web automation and interaction capabilities, particularly focused on Discord and other social platforms. The extension includes a comprehensive browser automation system with CSS selector generation, element interaction recording, and tab synchronization features. Based on the file structure and code analysis, it serves as both a general-purpose web automation tool and a specialized Discord interaction enhancer, with capabilities for recording user actions, executing automated sequences, and managing multiple browser tabs programmatically.

## Architecture

The extension follows a standard Chrome extension architecture with distinct components: a background script (discodancer/src/pages/Background/index.js) handles tab management and external API communication, content scripts (discodancer/src/pages/Content/) inject functionality into web pages and provide CSS selector generation capabilities, and multiple UI components including popup (discodancer/src/pages/Popup/), options page, devtools panel, and new tab page. The architecture supports bi-directional communication between components using Chrome's messaging API, with shared state management defined in discodancer/src/pages/shared.ts. The system includes a sophisticated CSS selector generator (css-selector.js) and element interaction system with support for recording and playback of user actions.

## Getting Started

1. Ensure Node.js version >= 18 is installed. 2. Clone the repository and navigate to the discodancer/ directory. 3. Run 'npm install' to install all dependencies including React 18, Material-UI, and development tools. 4. Modify discodancer/src/manifest.json to customize extension name and permissions. 5. Run 'npm start' to start the webpack dev server with hot reload. 6. Open Chrome and navigate to chrome://extensions/, enable Developer mode. 7. Click 'Load unpacked extension' and select the generated 'build' folder. 8. For production build, run 'NODE_ENV=production npm run build' which will generate a packaged extension zip file in the zip/ directory. 9. The extension will be available in Chrome with popup interface, options page, and background automation capabilities. 10. Configure external API endpoints in background script if using tab synchronization features.
