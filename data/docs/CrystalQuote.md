# CrystalQuote

![Language](https://img.shields.io/badge/language-C%23-blue)

## Overview

CrystalQuote is a multi-platform insurance quoting system built with C# ASP.NET Web API backend, React Native mobile application, and automated web scraping capabilities. The system appears to integrate with insurance carriers like Progressive to automate quote generation and management processes. It provides both agent-facing web interfaces and mobile applications for insurance quote management, with automated scraping tools to extract data from carrier websites.

## Architecture

The system follows a three-tier architecture with a C# ASP.NET MVC/Web API backend (Crystal.Quote), a React Native mobile frontend (CrystalQuote.Mobile), and Node.js-based web scraping automation tools (Crystal.Scrape). The backend includes standard MVC controllers for Account, Agent, Coverage, Driver, and Email management with Swagger API documentation. The scraping module uses Nightmare.js and Phantom.js to automate interactions with insurance carrier websites. The mobile app uses Redux for state management and React Native Router Flux for navigation.

## Key Features

- Automated web scraping of Progressive insurance website using Nightmare.js with login automation and quote initiation
- RESTful API with Swagger documentation for insurance quote management
- React Native mobile application with Google Places integration and mapping capabilities
- Multi-state insurance quote support with product selection automation
- Agent management system with authentication and authorization
- Coverage and driver management through dedicated controllers
- Email integration for quote communication
- Cross-platform mobile support with Redux state management

## Getting Started

1. Backend Setup: Open Crystal.Quote solution in Visual Studio, restore NuGet packages, configure connection strings in web.config, and run the ASP.NET application. 2. Mobile Setup: Navigate to CrystalQuote.Mobile/crystalquotemobile, run 'npm install' to install dependencies including React Native 0.50.4, configure native dependencies for react-native-maps and other native modules, then run 'react-native run-android' or 'react-native run-ios'. 3. Scraping Setup: Navigate to Auto/Crystal.Scrape, run 'npm install' to install Nightmare.js and dependencies, update credentials in resources.js, then run 'npm run night' to test automation or use the progressive.js test files for carrier-specific scraping.
