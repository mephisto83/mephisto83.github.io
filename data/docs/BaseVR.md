# BaseVR

## Overview

BaseVR is an Unreal Engine 4/5 virtual reality project template that provides a foundational framework for VR development. The project serves as a starting point or base template for creating VR applications, containing core VR functionality, character controllers, and game modes specifically designed for virtual reality experiences. Based on the project structure and naming conventions, this appears to be a development framework that abstracts common VR implementation patterns into reusable Blueprint assets and configurations.

## Architecture

The project follows Unreal Engine's standard directory structure with VR-specific components organized in the Content/00_VRBase directory. The architecture centers around key Blueprint classes including VR_Character.uasset for player representation, VR_Pawn.uasset for basic VR entity control, VR_GameMode.uasset and VR_GameModeBase.uasset for game logic management, and VR_BP_Library.uasset for shared VR utility functions. The project includes Android build configurations with VR-specific icons and splash screens, suggesting multi-platform VR support. Configuration files like DefaultEngine-Oculus.ini indicate Oculus VR platform integration, while the modular content organization allows for easy extension and customization.

## Getting Started

1. Install Unreal Engine 4.25+ with VR development components and Android SDK if targeting mobile VR. 2. Clone the repository and open BaseVr.uproject in Unreal Engine Editor. 3. Enable required VR plugins in Edit > Plugins (likely OpenXR, Oculus VR, or SteamVR depending on target platform). 4. Configure project settings for your target VR platform using the provided DefaultEngine.ini or DefaultEngine-Oculus.ini configurations. 5. Open Content/00_VRBase/Maps/VRBaseTest.umap to access the test environment. 6. Customize VR_Character, VR_Pawn, and VR_GameMode blueprints according to your project requirements. 7. For Android deployment, ensure Android SDK is configured and use the pre-configured build settings in Build/Android directory. 8. Package the project for your target VR platform through File > Package Project menu.
