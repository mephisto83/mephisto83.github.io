# mathdash

![Language](https://img.shields.io/badge/language-TypeScript-blue)

## Overview

MathDash is a real-time competitive math game platform designed for elementary school students (grades 1-3). The application provides both single-player practice sessions and multiplayer battles where students compete head-to-head in solving arithmetic problems. Built with React and Firebase Realtime Database, it features anonymous authentication, live matchmaking, comprehensive leaderboard systems, and instant feedback mechanisms to gamify math learning and encourage student engagement.

## Architecture

The application follows a client-server architecture with React as the frontend and Firebase providing backend services. The main.tsx file establishes React Router routing to five core pages: App (home), Practice (solo mode), Quick (matchmaking), Arena (live battles), and Boards (leaderboards). The firebaseClient.ts module handles authentication and database connections, while three core libraries manage business logic: questions.ts generates grade-appropriate math problems, matchmaking.ts handles player queuing and match creation, and leaderboard.ts manages scoring across daily/weekly/all-time rankings. State management utilizes Zustand for client-side state and Firebase Realtime Database for persistent multiplayer state synchronization. The Layout.tsx and Navigation.tsx components provide consistent UI structure across all pages.

## Key Features

- Real-time 1v1 multiplayer math battles with automatic matchmaking based on operation type (add/sub/mul/div) and grade level
- Solo practice mode with unlimited questions and immediate feedback for skill building
- Comprehensive scoring system awarding 1 point per correct answer, 10 bonus points for match wins, and 5 points for draws
- Multi-tiered leaderboard system tracking daily, weekly, and all-time rankings with automatic point aggregation
- Grade-appropriate question generation with configurable difficulty ranges for each arithmetic operation
- Anonymous authentication system eliminating registration barriers while maintaining user sessions
- Automatic match termination after 10 questions with winner determination and result persistence
- Real-time game state synchronization using Firebase Realtime Database for seamless multiplayer experience
- Responsive UI with Tailwind CSS optimized for young learners
- Automated deployment pipeline with Firebase Hosting integration

## Getting Started

1. Clone the repository and run 'npm install' to install dependencies including React, Firebase, and Vite. 2. Create a Firebase project at console.firebase.google.com and enable Anonymous Authentication in the Authentication section. 3. Set up Firebase Realtime Database and deploy the included database.rules.json using 'firebase deploy --only database'. 4. Copy Firebase configuration values from Project Settings and create a .env.local file with VITE_FB_API_KEY, VITE_FB_AUTH_DOMAIN, VITE_FB_DB_URL, VITE_FB_PROJECT_ID, and VITE_FB_APP_ID environment variables. 5. Install Firebase CLI globally and run 'firebase init' to configure hosting for your project. 6. Start development server with 'npm run dev' or deploy to production using 'npm run deploy' or the provided deploy.sh script. 7. Configure Firebase Hosting settings in firebase.json to serve the Vite-built dist folder. 8. Optionally run the included test scripts (test-auth.sh, test-matches.sh, test-navigation.sh) to verify functionality.
