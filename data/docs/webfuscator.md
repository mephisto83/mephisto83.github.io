## Overview

Webfuscator is a web content obfuscation and aggregation tool that scrapes news articles from websites like Breitbart and RedState, processes the text using natural language processing to replace words with synonyms, and generates new HTML pages with the obfuscated content. The tool appears to be designed for content rewriting or creating derivative articles by automatically replacing key words (nouns, verbs, adjectives, adverbs) with their synonyms while maintaining the original structure and meaning. The project includes a complete pipeline from web scraping to content generation using a magazine-style HTML template.

## Architecture

The system follows a modular architecture with five main Python components: PageReader handles web scraping and HTML parsing using lxml and CSS selectors, SynonymReader fetches synonyms from thesaurus.com with JSON caching, SpacyReader performs natural language processing using the spaCy library to identify parts of speech, Webfuscator orchestrates the obfuscation process and template generation, and program.py serves as the main execution script. The workflow involves scraping articles, parsing content using CSS selectors defined in articleSelections and breitbartSelector configurations, processing text through spaCy for POS tagging, replacing words with cached synonyms, and generating final HTML using a magazine template located in the html/ directory.

## Key Features

- Web scraping with CSS selector-based content extraction from news sites
- Natural language processing using spaCy for part-of-speech tagging and sentence parsing
- Synonym replacement system with thesaurus.com integration and JSON caching (synonymcache.json)
- Template-based HTML generation using magazine-style layouts
- Batch processing of multiple articles with pagination support (webfusdata{}.js files)
- Content similarity analysis for optimal synonym selection
- Support for multiple news site configurations (RedState and Breitbart)
- Automatic link discovery and crawling within specified domains
- Text obfuscation while preserving original article structure and images

## Getting Started

1. Install Python dependencies: pip install lxml spacy requests cssselect unidecode. 2. Download spaCy English model: python -m spacy download en. 3. Create synonymcache.json file in the root directory for synonym caching. 4. Configure site-specific selectors in program.py by modifying articleSelections and breitbartSelector arrays. 5. Set output_path variable to your desired HTML output directory. 6. Place the magazine template files in the html/ directory structure. 7. Run python program.py to start the obfuscation process. 8. The system will generate obfuscated HTML files in the specified output directory along with webfusdata.js files for dynamic loading.
