## Overview

This is a comprehensive WebVR/AR application built with A-Frame and React that provides an immersive 3D environment for AI-powered image generation and manipulation. The application integrates with Stable Diffusion APIs, Civitai models, and various image search services to create a virtual reality interface for creating, editing, and managing AI-generated content. It features a sophisticated painting system with brush tools, layer management, and real-time collaboration capabilities, all rendered in a 3D space that users can navigate using VR controllers or traditional input methods.

## Architecture

The system follows a Firebase-based serverless architecture with a React frontend hosting A-Frame WebVR components and Firebase Cloud Functions handling backend processing. The main application (my-app/) contains the React frontend with extensive A-Frame custom components for 3D UI elements, while Firebase Functions (my-app/functions/) manage API integrations, user authentication, payment processing via Stripe, and AI image generation workflows. The architecture separates concerns with dedicated services for user management, purchasable items, checkout sessions, and layer prompt processing, all backed by Firestore for data persistence and Firebase Realtime Database for real-time updates.

## Key Features

- Immersive 3D/VR interface using A-Frame with custom components like civitai-item, gui-circle-loader, and gui-circle-timer for interactive UI elements
- AI image generation pipeline (useStableDiffusionPipeline.ts) with support for multiple models, LoRA configurations, and real-time processing status
- Comprehensive painting system with brush tools, layer management, and canvas manipulation using Fabric.js integration
- Civitai model integration (CivitaiModelLoraConfigurationComponent.tsx) allowing users to browse, select, and configure AI models and LoRA weights
- Multi-source image search capabilities integrating Pexels, Pixabay, and Google Custom Search APIs
- Real-time collaboration features with Firebase Realtime Database for synchronized updates across users
- Stripe payment integration for credit-based usage system with webhook processing
- Advanced 3D model loading and manipulation with support for FBX, GLB, and OBJ formats
- Custom A-Frame components for complex UI interactions including drop-downs, sliders, checkboxes, and interactive menus
- Firebase authentication with custom claims and role-based access control

## Getting Started

1. Install dependencies: Run 'npm install' in both root directory and my-app/ directory. 2. Install Firebase CLI: 'npm install -g firebase-tools' and login with 'firebase login'. 3. Configure Firebase: Set up Firebase project and update firebase.json, database.rules.json, and firestore.rules with your project settings. 4. Set environment variables: Create secrets.js in root directory with API keys for Stable Diffusion, Pexels, Google Custom Search, and Stripe. 5. Deploy Firebase Functions: Navigate to my-app/functions/ and run 'npm run deploy'. 6. Configure Stripe: Set up webhook endpoints pointing to your Firebase Functions URL for payment processing. 7. Start development server: In my-app/ directory run 'npm start' to launch the React development server. 8. For production deployment: Run 'npm run deploy-prod' to build and deploy to Firebase Hosting. 9. SSL certificates: Place cert.pem and key.pem in root directory for HTTPS development if needed.
