## Overview

Tanner-Training is a cross-platform React Native fitness application designed for workout program management and exercise tracking. The app provides a comprehensive solution for creating custom workout programs, adding exercises with configurable parameters, and managing user fitness information. Built with Redux for state management, the application supports both iOS and Android platforms with native UI components from NativeBase, focusing on providing trainers and fitness enthusiasts with tools to structure and monitor their workout routines.

## Architecture

The application follows a Redux-based unidirectional data flow architecture with React Native Router Flux for navigation. The codebase is organized into distinct layers: Actions (appActions.js, programActions.js, uiActions.js, userInfoActions.js) handle business logic and API calls, Reducers (programsReducer.js, uiReducer.js, userInfoReducer.js) manage state transitions, and Screen components (home.js, login.js, createprogram.js, addexercise.js, userInfo.js) handle UI presentation. The app.js file serves as the root component configuring the Redux store with thunk middleware and defining the navigation structure. AsyncStorage middleware provides data persistence across app sessions, while a service utility layer handles external communications.

## Key Features

- Cross-platform mobile application supporting both iOS and Android with native build configurations
- Redux-powered state management with persistent storage using AsyncStorage for workout programs and user data
- Dynamic exercise creation system with configurable parameters including reps, weight, rest periods, and distance tracking
- Comprehensive workout program management allowing users to create, save, and modify custom training routines
- User profile management system for storing personal fitness information and preferences
- Navigation system using React Native Router Flux with scene-based routing between login, home, program creation, and exercise management screens
- Native UI components from NativeBase providing consistent cross-platform styling and user experience
- Modular action-reducer architecture enabling scalable state management for programs, UI interactions, and user information

## Getting Started

1. Clone the repository and navigate to the Tanner directory. 2. Install dependencies using 'npm install' to get all React Native and project-specific packages. 3. For Android development: Ensure Android SDK is installed, then run 'cd android && ./gradlew assembleDebug' to build the Android APK. 4. For iOS development: Open ios/Tanner.xcodeproj in Xcode and build the project with proper provisioning profiles. 5. Start the Metro bundler with 'npm start' to begin the React Native development server. 6. Run 'react-native run-android' or 'react-native run-ios' to launch the app on respective platforms. 7. For testing, use 'npm test' for Jest tests or 'npm run mocha' for Mocha test suite execution. 8. Configure the service endpoints in src/util/globals.js if connecting to external APIs for enhanced functionality.
