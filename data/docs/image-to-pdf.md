# image-to-pdf

![Language](https://img.shields.io/badge/language-TypeScript-blue)

## Overview

JustPDFit is a React-based web application that converts images to PDF format, built with TypeScript and deployed on Firebase. The application provides a client-side image-to-PDF conversion service using the pdf-lib library, allowing users to upload images and generate PDF documents directly in their browser without server-side processing. The project follows a modern React application structure with Firebase integration for hosting and analytics, targeting users who need quick and efficient image-to-PDF conversion capabilities.

## Architecture

The application follows a standard React single-page application architecture with TypeScript. The main conversion logic is encapsulated in the ImageToPDFConverter.tsx component, which likely handles file upload, image processing, and PDF generation using the pdf-lib library. Firebase is configured in Firebase.ts for hosting and analytics tracking. The application uses Create React App as the build system and development environment, with a typical src/ directory structure containing components, styles, and configuration files. The build process generates static assets that are deployed to Firebase hosting via the 'prod' npm script.

## Getting Started

1. Clone the repository and navigate to the justpdfit directory. 2. Install dependencies using 'npm install' to install React, TypeScript, Firebase, pdf-lib, and testing libraries. 3. Configure Firebase by updating the firebaseConfig object in src/Firebase.ts with your project credentials (currently contains placeholder values). 4. Run 'npm start' to launch the development server on localhost:3000. 5. For production deployment, ensure Firebase CLI is installed and authenticated, then run 'npm run prod' which builds the application and deploys to Firebase hosting. 6. Run 'npm test' to execute the Jest test suite. The application will be accessible via the Firebase hosting URL after deployment.
