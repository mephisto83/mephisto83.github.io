# wecanfeel

![Language](https://img.shields.io/badge/language-TypeScript-blue)

## Overview

WeCanFeel is a real-time collaborative cross-platform collage builder that enables multiple users to work simultaneously on shared canvases through web browsers and mobile devices. The application uses Firebase Realtime Database for synchronization and anonymous authentication, allowing users to join collaborative sessions via 4-character room codes without requiring signup. Built as a TypeScript monorepo with Yarn workspaces, it provides drawing tools, image manipulation capabilities, and an action replay system to visualize the creation process.

## Architecture

The project follows a monorepo architecture with three main packages: @wecanfeel/shared containing common TypeScript types and Firebase configuration, @wecanfeel/web implementing the React web application with Vite and React-Konva for canvas rendering, and @wecanfeel/mobile providing the React Native mobile app using Expo and React Native SVG. Firebase Realtime Database serves as the backend, storing board data including user sessions, drawing actions, images, and collaborative state. The database.rules.json defines security rules, while firebase.json configures hosting and database settings. Cross-platform compatibility is achieved through shared business logic in the shared package and platform-specific rendering implementations.

## Key Features

- Real-time collaborative drawing with multiple users on shared canvases
- Cross-platform support for web browsers and mobile devices (iOS/Android)
- Anonymous authentication system requiring no user registration
- Room-based sessions accessible via 4-character codes
- Drawing tools with customizable pen, brush, and eraser options
- Image drag-and-drop with resize and rotation capabilities
- Action replay system to visualize creation timeline
- Firebase-powered synchronization for seamless collaboration

## Getting Started

1. Clone the repository and navigate to the wecanfeel directory. 2. Install dependencies using 'yarn install' (requires Node.js 18+ with some dependencies needing Node.js 20+). 3. Build the shared package with 'yarn workspace @wecanfeel/shared build'. 4. Configure Firebase by updating packages/shared/src/firebase.ts with your Firebase project credentials including apiKey, authDomain, databaseURL, and projectId. 5. Set up Firebase Realtime Database and configure security rules using the provided database.rules.json file. 6. For web development, run 'yarn dev:web' to start the React application with Vite. 7. For mobile development, run 'yarn dev:mobile' to start the Expo development server. 8. Build production versions using 'yarn build:web' for web deployment and 'yarn build:mobile' for mobile app compilation.
