# square-the-images

![Language](https://img.shields.io/badge/language-TypeScript-blue)

## Overview

Square-the-Images is a TypeScript-based tool designed for processing and generating AI comic/manga content with character pose management and image generation workflows. The application processes story files containing character descriptions, scenes, and comic page layouts, then generates comprehensive configuration files for AI image generation systems. It specializes in mapping character poses to comic cells, managing AI model configurations, and standardizing sampler/scheduler parameters across different AI generation frameworks.

## Architecture

The application follows a modular architecture with four core TypeScript modules: index.ts serves as the main orchestrator that reads story JSON files and processes comic layouts; process.ts handles parsing of AI generation metadata and prompt extraction; model_info.ts contains a comprehensive database of AI model configurations with generation parameters; and poses.ts manages character pose definitions. The system transforms input story files (like pigstory.json) and comic layouts (comics_text_03_pig.json) into processed output files with standardized AI generation parameters. A complementary Python utility (squareit.py) provides image collage functionality using PIL for creating square grid layouts from generated images.

## Key Features

- Character pose mapping system that automatically assigns poses from a predefined library to comic cells based on story requirements
- AI model configuration standardization that converts between different sampler types (DPM++, Euler, Karras) and scheduler formats across generation frameworks
- Story-driven comic generation workflow that processes character descriptions, scenes, and page layouts from JSON story files
- Comprehensive AI model database (model_info.ts) containing generation parameters for 20+ different AI models including Stable Diffusion variants
- Prompt engineering utilities that parse and separate positive/negative prompts from AI generation metadata
- Image collage generation tool (squareit.py) that creates square grid layouts from folders of generated images
- Automated character setup that extracts character descriptions from story files and generates appropriate AI generation configurations

## Getting Started

1. Clone the repository and install dependencies with 'npm install'. 2. Ensure you have TypeScript installed globally or use the local devDependency. 3. Place your story JSON file (following the pigstory.json format) in the root directory. 4. Update the file paths in src/index.ts: set story_file, input_file, and output_file variables to point to your story file, input comic layout, and desired output location. 5. Build the TypeScript project using 'npm run build' which compiles to the dist/ directory. 6. Run the application with 'npm start' or directly with 'node dist/index.js'. 7. For image collaging functionality, ensure Python and PIL are installed, then modify the folder_path variable in squareit.py to point to your images directory and run the Python script. 8. The application will process your story file and generate an output JSON with standardized AI generation parameters and pose mappings.
