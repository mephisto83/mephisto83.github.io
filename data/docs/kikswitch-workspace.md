## Getting Started

1. Clone the repository from https://github.com/mephisto83/kikswitch-workspace
2. Install dependencies using 'npm install' or 'yarn install'
3. Start the development server with 'npm start' or 'yarn start' - this will open the application at http://localhost:3000
4. For production build, run 'npm run build' or 'yarn build' - this will create an optimized build in the build folder and execute the custom fixreport.js script
5. Run tests using 'npm test' or 'yarn test' to launch the Jest test runner
6. The application includes extensive CSS theming - customize the AdminLTE themes by modifying files in public/css/skins/ directory
7. TypeScript definitions for CSS modules are automatically generated - import CSS classes using the corresponding .d.ts files
8. For diagram functionality, refer to @projectstorm/react-diagrams documentation for creating custom node types and diagram behaviors
