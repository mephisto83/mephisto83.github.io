# red-react-phone-input

![Language](https://img.shields.io/badge/language-CSS-blue)

## Overview

red-react-phone-input is a highly customizable React component library for international phone number input with automatic formatting. It's a fork of react-phone-input-2 that provides a dropdown country selector, automatic phone number formatting based on country-specific patterns, and comprehensive internationalization support. The component handles complex phone number validation, formatting, and provides extensive customization options for styling and behavior, making it suitable for applications requiring robust international phone number input functionality.

## Architecture

The architecture follows a modular design with TypeScript as the primary language. The core component is defined in src/index.tsx which utilizes CountryData.ts for managing country information and phone number formats. The component leverages rawCountries.ts and rawTerritories.ts files containing comprehensive country data including dial codes, formatting patterns, and regional information. The styling system is built with LESS and provides multiple themes (material, bootstrap, semantic-ui, plain) defined in src/style/. The build system uses Webpack with TypeScript compilation, and includes utility scripts (move.js, pack-up.js) for build automation and version management.

## Key Features

- International phone number formatting with country-specific patterns
- Comprehensive country dropdown with flag icons and search functionality
- Multiple pre-built UI themes (material, bootstrap, semantic-ui, plain, high-res)
- Area code support for countries like US, Canada, and Argentina
- Territory support for dependent territories and smaller regions
- Extensive customization options for styling, behavior, and validation
- Built-in localization support for 11 languages (Spanish, German, Russian, French, Japanese, Chinese, Portuguese, Italian, Iranian, Arabic, Turkish, Indonesian)
- Custom mask support for phone number formatting
- Region-based country filtering (America, Europe, Asia, Oceania, Africa)
- TypeScript definitions for full type safety
- Debounced input handling and memoized country data for performance

## Getting Started

1. Install the package: `npm install red-react-phone-input --save`
2. Import the component and CSS: `import PhoneInput from 'red-react-phone-input'` and `import 'red-react-phone-input/lib/style.css'`
3. For development, clone the repository and run `npm install` to install dependencies
4. Use `npm run start` for JavaScript development server or `npm run start:css` for CSS development
5. Run tests with `npm test` using Jest
6. Build the project with `npm run build` which compiles TypeScript, processes LESS styles, and runs utility scripts
7. For custom styling, choose from available themes: style.css, material.css, bootstrap.css, semantic-ui.css, plain.css, or high-res.css
8. The component requires React 16.12.0+ and React-DOM as peer dependencies
