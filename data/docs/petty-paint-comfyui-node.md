# petty-paint-comfyui-node

> An integration between comfy ui and petty paint

![Language](https://img.shields.io/badge/language-Python-blue) ![Stars](https://img.shields.io/github/stars/mephisto83/petty-paint-comfyui-node)

## Overview

petty-paint-comfyui-node is a Python-based integration plugin that bridges the functionality between ComfyUI (a node-based UI for stable diffusion workflows) and Petty Paint (an image editing application). The project provides custom ComfyUI nodes that extend the platform's capabilities with specialized image processing, ControlNet masking, LoRA management, and enhanced image saving functionality. This integration allows users to seamlessly incorporate Petty Paint workflows into ComfyUI's node-based image generation pipeline, enabling more sophisticated image manipulation and AI-assisted painting workflows.

## Architecture

The project follows a modular architecture with each Python file implementing specific ComfyUI node classes. The core components include PettyPaint.py as the main integration module, PettyPaintControlNetToMasking.py for pose detection and body part masking using predefined color mappings, PettyPaintKSamplers.py for custom sampling operations with VAE encoding and alpha channel manipulation, PettyPaintLoras.py for LoRA (Low-Rank Adaptation) stack management, and specialized image handling modules (PettyPaintImageSave.py and PettyPaintImageStore.py). The utils.py module provides utility functions for file path management and image discovery. All components integrate with ComfyUI's node system through standardized INPUT_TYPES, RETURN_TYPES, and FUNCTION definitions.

## Key Features

- ControlNet pose detection with color-coded body part masking for head, torso, and legs using predefined RGB color mappings in PettyPaintControlNetToMasking.py
- Custom K-sampler implementations with VAE encoding, white-to-alpha conversion, and noise masking capabilities in PettyPaintKSamplers.py
- Advanced LoRA stack management supporting up to 3 concurrent LoRAs with individual weight controls and override capabilities in PettyPaintLoras.py
- Flexible image saving system with multiple format support (PNG, JPG, WEBP, BMP, TIFF) and quality controls in PettyPaintImageSave.py and PettyPaintImageStore.py
- File path utilities for recursive image discovery and model management through the PPApi class in utils.py
- Integration with ComfyUI's node system with proper input validation, type checking, and UI category organization

## Getting Started

1. Install ComfyUI and ensure it's properly configured. 2. Clone this repository into your ComfyUI custom_nodes directory. 3. Install the required dependencies using 'pip install -r requirements.txt' (Flask==2.1.2 and other ComfyUI dependencies). 4. Ensure you have the necessary ComfyUI modules and folder_paths configured for LoRA and model access. 5. Restart ComfyUI to load the new PettyPaint nodes. 6. The nodes will appear under the 'PettyPaint' category in the ComfyUI interface. 7. Configure file paths in the image save nodes to point to your desired output directories. 8. For ControlNet functionality, ensure you have the appropriate ControlNet models installed in your ComfyUI models directory.
