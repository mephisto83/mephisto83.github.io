## Overview

Composer Companion is a comprehensive web-based music composition and education platform that provides interactive musical instruments, MIDI playback capabilities, and music theory tools. Built as a Firebase-hosted application, it combines real-time audio synthesis using Web Audio API with extensive musical notation rendering through VexFlow and ScoreLibrary integration. The platform serves as both a learning tool for musicians and a creative environment for composition, featuring virtual instruments like piano, guitar, violin, and various percussion instruments alongside advanced music visualization and playback features.

## Architecture

The application follows a client-side web architecture with Firebase backend services. The frontend is built around a modular JavaScript structure with separate instrument modules (public/js/piano.js, public/js/guitar.js, public/js/violin.js, etc.) that utilize Web Audio API for sound synthesis. Music notation is handled by two primary libraries: VexFlow (public/Vexflow/trunk/) for dynamic notation rendering and ScoreLibrary (public/score-library/) for MusicXML processing. The public/Data/ directory contains chord progressions, voice definitions, and musical font assets, while public/js/midi/ provides MIDI playback functionality. Firebase Functions (functions/index.js) handle server-side operations, and the application uses CodeMirror plugins for code editing functionality related to music programming.

## Key Features

- Interactive virtual instruments including piano, guitar, violin, harp, kalimba, timpani, and xylophone with realistic audio synthesis
- Real-time MIDI playback and recording capabilities through public/js/midi/ modules
- Advanced music notation rendering using VexFlow library with support for complex musical symbols and accidentals
- MusicXML score parsing and display through ScoreLibrary integration
- Comprehensive chord progression database stored in public/Data/chord_data.js
- Voice and instrument sample management with multiple audio format support
- Interactive metronome with web worker implementation (public/js/metronomeworker.js)
- Visual waveform display using SiriWave library for audio feedback
- CodeMirror-based code editor with syntax highlighting for various music programming languages
- Microphone input processing for real-time audio analysis
- Responsive design with Pace.js loading animations in multiple color themes

## Getting Started

1. Clone the repository and ensure Node.js 14+ is installed. 2. Install Firebase CLI globally: `npm install -g firebase-tools`. 3. Navigate to the functions directory and install dependencies: `cd functions && npm install`. 4. Configure Firebase project settings in firebase.json and ensure proper authentication. 5. Deploy Firestore security rules: `npm run publish-rules`. 6. For local development, start Firebase emulators: `cd functions && npm run serve`. 7. Deploy the complete application: `npm run publish`. 8. Ensure all audio assets in public/Data/ are properly uploaded and accessible. 9. Configure MIDI.js by running the build script in public/Midi/trunk/: `./build.sh`. 10. Test virtual instruments by accessing the hosted application and verifying Web Audio API compatibility in target browsers.
