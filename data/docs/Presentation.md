## Overview

This is a comprehensive Blender addon suite designed for automated presentation and animation generation. The project provides tools for creating dynamic 3D presentations, automated ship/fleet generation, and sophisticated composite scene management through JSON-driven workflows. The system enables users to define complex 3D scenes, animations, and rendering pipelines through configuration files rather than manual Blender operations, making it particularly valuable for automated content generation and batch processing workflows.

## Architecture

The architecture follows a modular plugin system with several interconnected components: PresentationBlender.py serves as the main addon entry point, BlenderToJson.py handles scene serialization and keyframe extraction, CompositeWriter.py manages node-based compositing workflows, FleetMaker.py provides procedural 3D asset generation, and CompositeRecipes.py offers a recipe-based system for reusable composite configurations. The system uses Constants.py for shared configuration mappings between Blender's API and JSON representations. Installation is managed through InstallPresentation.py and FleetInstall.py scripts that handle addon deployment and dependency management across different platforms.

## Key Features

- JSON-driven scene configuration and animation keyframe management through BlenderToJson.py
- Automated procedural ship/fleet generation with UV unwrapping and OBJ export via FleetMaker.py
- Node-based compositing system with recipe management for reusable composite configurations
- Cross-platform installation system supporting Windows and Linux environments
- Integration with professional lighting systems including Pro Lighting Studio and Pro Skies
- Automated batch processing with support for frame-by-frame rendering and animation export
- Material and texture management with support for EXR textures and custom shader workflows
- Timeline marker extraction and JSON export for presentation synchronization

## Getting Started

1. Run installPresentation.bat on Windows or execute the Python installation scripts directly. 2. For automated installation, use 'blender -b --python InstallPresentation.py' to install the core PresentationBlender addon. 3. For fleet generation capabilities, use 'blender -b --python FleetInstall.py' with optional parameters for Blender version and installation path. 4. Ensure Blender version 2.79+ is installed and accessible. 5. The system will automatically enable required addons including io_import_images_as_planes, and optionally pro_lighting_studio and pro_lighting_skies_ultimate. 6. Configure composite_recipe_directory in the Blender scene properties to point to your JSON recipe files location (default: D:\BlenderRecipes\). 7. Place .blend files in the blends/ directory and configuration JSON files in test_configs/ for testing. 8. Use the Node Editor tools panel to access Composite Recipes functionality for applying JSON-defined node trees.
