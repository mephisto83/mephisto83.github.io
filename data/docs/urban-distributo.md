# urban-distributo

![Language](https://img.shields.io/badge/language-TypeScript-blue)

## Overview

urban-distributo is a TypeScript-based web scraper specifically designed to extract definitions from Urban Dictionary. The tool uses Puppeteer for browser automation and implements a custom concurrency limiter to manage multiple scraping tasks efficiently. The scraper navigates through Urban Dictionary pages, extracts word definitions with their meanings and examples, and can recursively discover new URLs to continue the scraping process while maintaining performance through optimized browser instances and resource blocking.

## Architecture

The application follows a single-threaded, event-driven architecture built around Puppeteer's browser automation. The core module urban.ts implements a custom rate-limiting system using createPLimit() function to control concurrent browser instances. Each scraping task launches its own browser instance with optimized settings, blocks unnecessary resources (images, stylesheets, fonts), and uses random user agents for stealth. The scrapePage() function extracts data using CSS selectors targeting '.definition', '.word', '.meaning', and '.example' elements, while scrapeURL() manages the browser lifecycle and data processing.

## Key Features

- Concurrent web scraping with custom rate limiting through createPLimit implementation
- Resource optimization by blocking images, stylesheets, and fonts during page loads
- Random user agent rotation to avoid detection using getRandomUserAgent()
- Structured data extraction returning DefinitionData interface with word, meaning, example, and source URL
- Comprehensive logging system using Winston with both console and file output to scraper.log
- Recursive URL discovery within definition pages for extended crawling capabilities
- Automatic version management through npm scripts with patch increments on build

## Getting Started

1. Clone the repository and navigate to the project directory. 2. Install dependencies using 'npm install' which will automatically trigger the prepare script and build the project. 3. The build process compiles TypeScript files from src/ to dist/ directory using the tsconfig.json configuration. 4. Run the scraper using 'npm run go' which builds the project and executes node dist/urban with a sample Urban Dictionary URL. 5. Alternatively, build manually with 'npm run build' and then execute 'node dist/urban.js [URL]' with your target Urban Dictionary page URL. 6. Monitor the scraping process through console output or check scraper.log for detailed logging information.
