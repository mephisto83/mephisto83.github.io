# react-native-social-identity-login

> React Native component to login to social media

![Language](https://img.shields.io/badge/language-Objective-C-blue)

## Overview

react-native-social-identity-login is a React Native component library designed to facilitate social media authentication in mobile applications. The project provides a reusable WebView-based login solution that handles OAuth flows for social platforms by intercepting URL redirects and extracting access tokens from hash parameters. Built in 2016-2018, this library offers a simple integration path for developers who need to implement social login functionality without dealing directly with platform-specific SDKs.

## Architecture

The architecture consists of a main React Native component (ReactNativeLogin) that wraps a WebView for OAuth authentication flows. The core component in social-login/index.js manages authentication state through React lifecycle methods and URL navigation monitoring. The iOS native app structure includes standard React Native boilerplate with AppDelegate.m handling app initialization and RCTRootView setup. The component uses a state-driven approach where it transitions between initial loading, authentication WebView, and logged-in states based on URL parsing and access token extraction. The authentication flow relies on monitoring WebView navigation events and parsing OAuth callback URLs to extract access tokens from hash fragments.

- Configurable login and initial view components through props
- State management for authentication lifecycle (loading, authenticating, logged-in)
- Cross-platform support with iOS native implementation included
- Cookie handling integration via react-native-cookies dependency
- Customizable UI through loginView and initialview props
- Navigation state monitoring for OAuth callback detection

## Getting Started

1. Install dependencies by running 'npm install' in the project root directory. 2. Ensure React Native 0.36.1 and React 15.3.2 are properly installed as specified in package.json. 3. For iOS development, open ios/reactnativesocialidentitylogin.xcodeproj in Xcode and configure signing certificates. 4. Import the ReactNativeLogin component from social-login/index.js in your application. 5. Configure the component by passing a loginUrl prop with your OAuth provider's authorization URL. 6. Optionally provide custom loginView and initialview components through props. 7. Implement the onAuthentification callback prop to handle successful authentication and access token retrieval. 8. Run 'react-native run-ios' to build and deploy the application. 9. Test the OAuth flow by navigating to the configured social login URL and verifying access token extraction from callback URLs.
