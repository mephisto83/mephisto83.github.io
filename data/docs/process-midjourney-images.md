# process-midjourney-images

![Language](https://img.shields.io/badge/language-TypeScript-blue)

## Overview

Process Midjourney Images is a comprehensive TypeScript-based toolkit designed to automate and streamline the processing of AI-generated images from Midjourney. The project provides a multi-faceted solution that combines image processing capabilities, Discord bot integration, web automation for e-commerce platforms like Contrado, and workflow automation for creative content production. It serves as a bridge between AI image generation and practical application in commercial contexts, offering tools for image manipulation, batch processing, social media automation, and print-on-demand product creation.

## Architecture

The project follows a modular architecture with distinct layers for different functionalities. The core image processing engine is built around Sharp and Jimp libraries, handling operations like cropping, resizing, and collage creation through src/index.ts and related modules. A sophisticated workflow automation system is implemented through src/recipeProcessing.ts and the src/recipes/ directory, which uses JSON-based recipe files to define automated browser interactions. The Discord integration layer (src/discord.js) provides bot functionality for image management, while Express-based web servers (src/app.ts, src/local_server.js) offer REST APIs for image operations. Python scripts in the python/ directory handle specialized tasks like background removal using rembg and automatic1111 integration. The recipe system creates a domain-specific language for web automation, with TypeScript interfaces generated from JSON recipe definitions.

## Getting Started

1. Install Node.js 18 and Python 3.x on your system. 2. Clone the repository and run 'npm install' to install TypeScript dependencies including Sharp, Express, and Discord.js. 3. Install Python dependencies by navigating to python/rembg/ and installing rembg library for background removal. 4. Configure Firebase credentials for cloud integration if needed. 5. Set up directory structure with input folders (default: F:\midjourney\letters\in) and output directories as specified in src/index.ts config object. 6. Build the TypeScript project using 'npm run build' which compiles to lib/ directory. 7. Configure Discord bot tokens and Contrado credentials for automation features. 8. Run specific workflows using npm scripts like 'npm run midjourney' for image processing, 'npm run discord' for bot functionality, or 'npm run workflow' for recipe automation. 9. For web automation, ensure Chrome/Chromium is installed for browser automation scripts. 10. Set up automatic1111 integration by copying scripts from python/automatic1111/scripts/ to your Stable Diffusion installation if using AI image generation features.
