# MathPractice

![Language](https://img.shields.io/badge/language-Objective-C-blue)

## Overview

MathPractice is a React Native mobile application designed to render mathematical expressions on both iOS and Android platforms. The application leverages KaTeX, a fast math typesetting library, to display properly formatted mathematical equations within a WebView component. Created as a cross-platform solution, it demonstrates how to integrate web-based math rendering libraries into native mobile applications for educational or mathematical content display purposes.

## Architecture

The application follows a standard React Native architecture with a WebView-based rendering approach. The main App.js component renders a WebView that loads template.html, which contains KaTeX library integration for mathematical expression rendering. The template.js file exports an HTML string template that includes KaTeX CDN resources and renders a sample mathematical equation (c = ±√(a² + b²)). The project maintains separate iOS (Objective-C/Xcode) and Android (Java/Gradle) native implementations, with the React Native bridge handling cross-platform communication. The architecture separates concerns by keeping the math rendering logic in web technologies while maintaining native mobile app performance and capabilities.

## Getting Started

1. Clone the repository and navigate to the mathpractice directory. 2. Install dependencies using 'npm install' or 'yarn install' based on the yarn.lock file present. 3. For iOS development: Navigate to ios/ directory, open mathpractice.xcodeproj in Xcode, and ensure proper signing certificates are configured. 4. For Android development: Ensure Android SDK is installed and ANDROID_HOME environment variable is set, then use the gradle wrapper in android/ directory. 5. Start the Metro bundler with 'npm start' or 'yarn start'. 6. Run the iOS app using 'react-native run-ios' or Android app using 'react-native run-android'. 7. The app will load template.html through the WebView component in App.js, which renders the KaTeX mathematical expression defined in template.js. 8. Run tests using 'npm test' or 'yarn test' to execute the Jest test suite defined in __tests__/App.js.
