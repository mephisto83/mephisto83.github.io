## Overview

This is a React-based demonstration project that implements a positionable UI component system, allowing users to interactively manipulate DOM elements through dragging, rotating, scaling, and z-index control. The project appears to be built as a proof-of-concept or demo, combining a Bootstrap-based business website template with custom React components that enable real-time visual manipulation of elements. The core value proposition is showcasing how CSS3 transforms can be controlled through React components to create an interactive design tool or content management interface.

## Architecture

The application follows a Redux-based React architecture with the main entry point in main.js importing src/app.js. The Redux store is configured with reducers from src/reducers.js and uses redux-thunk middleware for async actions. The core positionable functionality is implemented through a set of specialized controller components: PlacementController for drag movement, RotationController for rotation, ScaleController for scaling, and ZIndexController for layering. These controllers utilize Transform-mixin and Scale-mixin for shared transformation logic. The main Positionable component in src/positionable/Positionable.js orchestrates these controllers and uses react-onclickoutside to handle deactivation when clicking outside elements. The UI is rendered through src/components/app.js which combines the Bootstrap business template with the positionable components.

## Key Features

- Interactive element positioning through mouse drag operations
- Real-time rotation control with visual feedback
- Dynamic scaling with proportional control counter-scaling
- Z-index manipulation for element layering
- Click-outside deactivation using react-onclickoutside
- Visual control handles that appear when elements are activated
- CSS3 transform-based animations for smooth interactions
- Redux state management for component state persistence
- Modular controller architecture with mixins for shared functionality
- Integration with Bootstrap UI framework for responsive design

## Getting Started

1. Clone the repository and navigate to the project directory. 2. Install dependencies using 'npm install' which will install React, Redux, Babel presets, and other required packages listed in package.json. 3. Start the development server using 'npm run live-server' to serve the static HTML files. 4. For development with auto-compilation, run 'npm run watchify' to automatically rebuild bundle.js when source files change. 5. Alternatively, use 'npm run browserify' for a one-time build of the React components into bundle.js. 6. Open index.html in your browser to see the Business Casual template with the positionable component demo. The positionable element can be activated by clicking on it, which will display drag handles, rotation controls, and scaling options.
