## Overview

red-auto-ml is an experimental machine learning repository that serves as a learning playground for TensorFlow and computer vision applications. The project contains two primary demonstrations: a basic neural network for fashion item classification using the Fashion-MNIST dataset, and an object detection system utilizing pre-trained COCO models. This appears to be a personal educational project focused on exploring automated machine learning workflows and web-based ML implementations, though the web component is not yet implemented in the current codebase.

## Architecture

The project follows a simple script-based architecture with two main components. The basic-classification.py module implements a sequential neural network using TensorFlow/Keras with a flattened input layer, dense hidden layer with ReLU activation, and softmax output layer for 10-class classification. The object_detection_demo.py module leverages TensorFlow's Object Detection API, downloading pre-trained SSD MobileNet models and implementing inference pipelines with visualization capabilities. Both modules are standalone scripts that can be executed independently, with hardcoded paths pointing to TensorFlow models research directory structure.

## Key Features

- Fashion-MNIST classification with 94%+ accuracy using a 3-layer neural network
- Real-time prediction visualization with confidence scores and color-coded accuracy indicators
- Pre-trained object detection using SSD MobileNet v1 COCO model with automatic model downloading
- Batch image processing with bounding box visualization and label mapping
- Interactive plotting capabilities with matplotlib for model performance analysis
- Support for both CPU and GPU TensorFlow execution environments

## Getting Started

1. Create and activate a Python virtual environment: `source ./venv/bin/activate` 2. Install TensorFlow GPU support: `pip install tensorflow-gpu` 3. Install visualization dependencies: `pip3 install matplotlib` and `sudo apt-get install python3-tk` 4. Clone TensorFlow models repository to `~/dev/tensorflow/models/` 5. Install COCO API by copying pycocotools to `~/dev/tensorflow/models/research/` 6. Update hardcoded paths in object_detection_demo.py to match your TensorFlow models installation directory 7. Ensure test images are available in `/home/andrewp/dev/tensorflow/models/research/object_detection/test_images/` 8. Execute basic-classification.py for fashion classification demo or object_detection_demo.py for object detection demo
