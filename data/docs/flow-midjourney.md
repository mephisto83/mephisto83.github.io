## Overview

Flow-Midjourney is a comprehensive AI-powered image processing and machine learning workflow management system built on Node.js and Express.js. The platform orchestrates complex computer vision tasks including YOLO model training, image generation, email automation, and browser-based image processing. It integrates with Google Cloud Platform for scalable GPU-accelerated computing, Firebase for data management, and various AI services including OpenAI and Groq for intelligent content processing.

## Architecture

The system follows a microservices architecture with a central Express.js server (server.js) coordinating multiple specialized modules. Core components include a YOLO training pipeline (trainer/yolo-job-processor.py), Gmail polling system (gmail-poller/), browser automation utilities (browser-utils/), and GPU-enabled worker deployments. The architecture supports both Cloud Run serverless deployments and VM-based GPU instances for intensive ML workloads. Data flows through Firebase Admin SDK for persistence, with image processing handled by Canvas and Sharp libraries, while GPU workloads utilize PyTorch with CUDA support.

## Key Features

- GPU-accelerated YOLO model training with A100/H100 support via trainer/yolo-job-processor.py
- Intelligent Gmail email processing and filtering with automated responses using gmail-poller/ and filter-gmail-poller/
- Multi-modal AI integration supporting OpenAI GPT, Groq, and custom vision models
- Browser automation for web scraping and image capture using browser-utils/utils.js and robotjs
- Scalable cloud deployment with automated worker pool management via deploy-worker-pools.sh
- Real-time image processing pipeline with Canvas, Sharp, and FFmpeg integration
- Firebase-integrated data management with automatic backup and synchronization
- Container orchestration supporting both CPU and GPU workloads with Docker multi-stage builds
- RESTful API endpoints for job management, configuration, and health monitoring

## Getting Started

1. Install dependencies: npm install (includes PyTorch CUDA support via Dockerfile). 2. Configure Firebase: Place flowframe-auto-firebase-adminsdk.json service account key in root directory. 3. Set up Gmail API: Run cd gmail-poller && node setup-auth.js for OAuth2 authentication. 4. Configure environment: Copy .env.example to .env and set required variables. 5. For local development: npm run dev to start with nodemon auto-reload. 6. For GPU deployment: npm run deploy-gpu-vm to create A100-enabled GCP VM, or npm run workers:deploy:gpu-l4 for Cloud Run GPU workers. 7. Verify setup: Access http://localhost:8080/health for health check and /api/config for configuration status. 8. Optional: Run npm run train_processor for YOLO model training if ML workloads are required.
