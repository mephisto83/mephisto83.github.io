# Client-Profile-Case-Study-App

![Language](https://img.shields.io/badge/language-TypeScript-blue)

## Overview

The Client Profile Case Study App is a comprehensive web application designed for educational institutions and organizations to manage client assessment through interactive character sheets and dimensional analysis. The platform enables facilitators to create structured assessment frameworks using character dimensions (questions with multiple choice options), organize them into character sheets, and assign these assessments to classes of students. Students can complete these assessments to generate character profiles, while administrators can manage users, track progress, and export results for analysis.

## Architecture

The application follows a modern three-tier architecture with a React-TypeScript frontend (web/src), Firebase backend services, and cloud functions for server-side logic. The frontend uses atomic design patterns with components organized into Layout/, Pages/, and UI/ directories. State management is handled through React Context API with custom hooks for Firebase integration. The backend leverages Firestore for real-time data storage with collections for users, organizations, character dimensions, character sheets, and class management. Firebase Cloud Functions (functions/index.js) provide server-side operations like user role assignment, password resets, PDF generation, and email notifications. The data flow follows: Firebase Auth → User authentication → Firestore real-time sync → React Context → Component rendering, with TypeScript interfaces (types/index.ts) ensuring type safety across all layers.

## Key Features

- Role-based authentication system with admin, facilitator, and student roles managed through Firebase Auth and Firestore user collections
- Character dimension management allowing facilitators to create assessment questions with multiple choice options and tagging system (CharacterDimensionCard.tsx, DimensionOptionsEditor.tsx)
- Dynamic character sheet builder that combines multiple dimensions into structured assessments with class assignment capabilities (CharacterSheetPage.tsx, ClassAssignmentModal.tsx)
- Class management system with QR code generation for easy student enrollment and bulk user management (ClassCard.tsx, QRCodeComponent.tsx)
- Real-time assessment completion tracking with CharacterSheetInstance documents storing student responses and timestamps
- Comprehensive admin dashboard with user management, organization statistics, and bulk operations (UserManagementPage.tsx, StudentsPage.tsx)
- GDPR compliance features including cookie consent management, data export functionality, and account deletion with personal data removal (GDPRCompliance.tsx, CookieConsent.tsx)
- PDF and JSON export capabilities for assessment results through Firebase Cloud Functions
- Dark/light theme support with system preference detection and persistent storage
- Responsive design with Tailwind CSS ensuring cross-device compatibility

## Getting Started

1. Clone the repository and navigate to the project directory. 2. Install Firebase CLI globally: 'npm install -g firebase-tools'. 3. Set up Firebase project: run 'firebase login' and 'firebase init' to configure hosting, functions, and Firestore. 4. Install web dependencies: 'cd web && npm install'. 5. Install function dependencies: 'cd functions && npm install'. 6. Configure Firebase environment by updating web/src/firebase/config.ts with your Firebase project credentials (apiKey, authDomain, projectId, etc.). 7. Set up Firestore security rules by deploying firestore.rules to your Firebase project. 8. Configure email services for Cloud Functions: 'firebase functions:config:set sendgrid.key=YOUR_API_KEY' and 'firebase functions:config:set email.from=your-email@domain.com'. 9. Deploy Firebase Functions: 'firebase deploy --only functions'. 10. Start local development server: 'cd web && npm run dev' which starts Vite dev server on localhost:5173. 11. For production deployment: run 'npm run build' from root to build the web app, then 'firebase deploy --only hosting' to deploy to Firebase Hosting.
