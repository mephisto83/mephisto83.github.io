# lyrics-extractor

> extracts lyrics from music

![Language](https://img.shields.io/badge/language-Python-blue)

## Overview

lyrics-extractor is a Python-based tool that combines audio transcription and 3D animation to create animated lyric videos. The system uses OpenAI's Whisper model to extract word-level timestamps from audio files, then generates synchronized 3D text animations in Blender where each word appears and disappears based on its timing in the original audio. The project consists of two main components: whisp.py for audio transcription and extractor.py for Blender-based animation generation.

## Architecture

The system follows a two-stage pipeline architecture. Stage 1 (whisp.py) uses the Whisper library to transcribe audio files into JSON format with word-level timestamps, creating structured data with segments containing individual words and their precise start/end times. Stage 2 (extractor.py) operates as a Blender script that reads the JSON transcription data and programmatically creates 3D text objects for each word, animating them to slide in and out of frame based on their timestamps. The extractor module handles material creation, font loading, keyframe animation, and rendering configuration, with support for both normal phrase-based animation and individual word animation modes.

## Key Features

- Word-level audio transcription using Whisper 'turbo' model with precise timestamps
- Automated 3D text animation generation in Blender with customizable slide-in/slide-out effects
- Configurable neon sign styling with material and font customization support
- Adaptive animation timing that handles both short and long duration words differently
- PNG sequence rendering with transparent backgrounds for video composition
- Text wrapping functionality for longer phrases with configurable line length limits
- Support for both individual word animation and phrase-based animation modes

## Getting Started

1. Install required dependencies: pip install whisper-openai and ensure Blender is installed with Python API access. 2. Place your audio file in the project directory and update the audio_path variable in whisp.py. 3. Run whisp.py to generate transcription: python whisp.py (this creates a JSON file with word timestamps). 4. Configure extractor.py by setting JSON_PATH to your transcription file, FONT_PATH to a valid system font, and OUTPUT_PATH for rendered frames. 5. Open Blender and run extractor.py as a script, or execute via command line: blender --background --python extractor.py. 6. The system will generate animated text objects and render PNG sequences to the specified output directory. 7. Adjust parameters like FPS, MIN_ANIM_FRAMES, NEON_SIGN, and USE_PHRASES based on your animation preferences.
