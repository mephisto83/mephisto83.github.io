# Playwatch

![Language](https://img.shields.io/badge/language-CSS-blue)

## Overview

Playwatch is a React-based streaming platform that provides video content and games for children with multiple subscription tiers. Originally built from a React-Redux-TypeScript template, the application has evolved into a comprehensive entertainment platform featuring user authentication, subscription management, video streaming capabilities, and child-safe content delivery. The platform offers three subscription tiers (Free, Play, and Premium) with varying levels of access to content and features, including parental controls, profile switching, and content curation for different age groups.

## Architecture

The application follows a Redux-based state management architecture with modular organization. The codebase is structured with src/app/modules/ containing feature-specific modules like login functionality with actions, reducers, services, and interfaces. The UI layer utilizes Material-UI components for consistent design, while Firebase integration handles backend services. The application uses React Router for navigation between different views including pricing pages, subscription management, and content streaming. Static assets are organized in public/ with extensive CSS styling including AdminLTE theme integration. The login module demonstrates a complete Redux pattern with login.actions.ts, login.reducer.ts, login.service.ts, and corresponding TypeScript interfaces.

## Key Features

- Multi-tier subscription system with Free, Play ($5.99/month), and Premium ($9.99/month) plans
- User authentication and authorization with JWT token management
- Child profile management with up to 3 child accounts on Play tier
- Video streaming capabilities with react-player integration
- Game access and interactive content delivery
- Station creation and sharing functionality
- Commercial-free experience on Premium tier
- Responsive design with mobile and desktop support
- Firebase backend integration for data persistence
- Form validation using Formik and Yup schemas
- Redux state management with TypeScript interfaces
- Axios interceptors for API request/response handling

## Getting Started

1. Clone the repository and navigate to the project directory. 2. Install dependencies using 'npm install' to install all packages listed in package.json including React, Redux, Material-UI, and Firebase. 3. Configure Firebase by setting up your Firebase project credentials and updating the configuration in the codebase. 4. Set up environment variables including REACT_APP_SERVER_URL for API endpoints as referenced in login.endpoints.ts. 5. Run 'npm start' to launch the development server on http://localhost:3000. 6. For production deployment, run 'npm run build' to create optimized build files. 7. Configure the homepage URL in package.json (currently set to https://mephisto83.github.io/) for your deployment target. 8. Use 'npm run prettier:write' to format TypeScript files and 'npm test' to run the test suite.
