## Overview

cheap-tickets is a TypeScript-based web scraping tool designed to automate flight price searches on travel booking websites, specifically Momondo. The application uses Puppeteer to control a headless Chrome browser, automatically filling search forms with flight parameters and initiating searches to help users find affordable airfare options. This project serves as a proof-of-concept for automated travel price comparison, allowing users to programmatically search for flights between specified airports within defined date ranges.

## Architecture

The application follows an object-oriented architecture with clear separation of concerns across four main TypeScript classes. The entry point (src/index.ts) initializes the CheapSearch class, which serves as the main orchestrator managing browser instances and search operations. The CheapSearch class maintains a collection of SearchPages instances, where each SearchPages object represents a browser tab configured with specific SearchParameters (origin, destination, departure/return dates). The SearchPages class handles all DOM manipulation and user interaction simulation, including form field population, button clicking, and search execution using CSS selectors to interact with Momondo's interface.

## Key Features

- Automated browser control using Puppeteer for headless web scraping
- Momondo integration with specific CSS selectors for form interaction (ORIGIN_CSS_SELECTOR, DESTINATION_CSS_SELECTOR, DEPARTURE_DATE_CSS)
- Multi-search capability through SearchPages collection management
- Configurable search parameters including airport codes, departure and return dates
- Built-in delays and wait mechanisms to handle dynamic content loading
- Form field manipulation including text input clearing, typing with realistic delays, and keyboard event simulation
- Date formatting and conversion from timestamps to MM/DD/YYYY format for travel booking forms

## Getting Started

1. Clone the repository and navigate to the project directory. 2. Install dependencies by running 'npm install' which will install Puppeteer and TypeScript type definitions. 3. Compile the TypeScript source code using 'tsc' command (TypeScript compiler configuration is defined in tsconfig.json). 4. Run the application using 'npm start' which executes 'tsc && node ./out/index.js'. 5. The application will launch a visible Chrome browser instance (headless: false) and automatically navigate to Momondo to search for flights from MSP to HNL with dates 10-100 days in the future. 6. Modify the search parameters in src/index.ts to customize origin/destination airports and date ranges as needed.
