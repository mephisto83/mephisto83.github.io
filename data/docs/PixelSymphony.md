# PixelSymphony

> Pixel Symphony

![Language](https://img.shields.io/badge/language-Python-blue)

## Overview

PixelSymphony is a Blender addon written in Python that provides a comprehensive framework for procedural animation and property management within Blender's 3D environment. The project implements a sophisticated system for creating dynamic animations through expression-based property manipulation, custom object properties, and automated rendering workflows. It appears to be designed for artists and developers who need fine-grained control over object transformations, material properties, and animation sequences through a node-based or expression-driven approach.

## Architecture

The addon follows a modular architecture organized under the pixel_orchestra package with distinct functional components. The core architecture includes: pixel_collection.py for managing Blender collection properties, pixel_custom_object_properties.py and pixel_custom_properties.py for handling dynamic property assignment with expression evaluation capabilities, pixel_rendering.py for automated frame rendering with change detection algorithms, pixel_stored_functions.py containing utility functions for property calculations, and pixel_symphony_nodes.py for node-based operations. The system uses a property naming convention with 'pix_' prefixes and supports expression evaluation through variables like 'var1' with corresponding property paths and expressions stored as custom properties.

## Key Features

- Custom property management system with automatic 'pix_' prefixed property creation and batch editing capabilities
- Expression-based animation system supporting variable binding and mathematical expressions for procedural property updates
- Intelligent frame rendering system with change detection algorithms that optimize rendering by only processing frames where object transformations or material properties have changed
- Collection-based organization with PIXEL_COLLECTION property management for grouping related objects
- Modal rendering operator for automated batch frame processing with timer-based execution
- Comprehensive object property coverage including location, rotation, scale, color (RGB), and brightness with full range numeric support
- Function library system with stored mathematical functions like velocity adjustment and parameter-based calculations
- Integration with Blender's native UI panels including custom property panels and collection properties

## Getting Started

1. Download or clone the PixelSymphony repository containing the pixel_orchestra folder. 2. Copy the entire pixel_orchestra folder to your Blender addons directory (typically ~/.config/blender/[version]/scripts/addons/ on Linux, %APPDATA%\Blender Foundation\Blender\[version]\scripts\addons\ on Windows). 3. Launch Blender and navigate to Edit > Preferences > Add-ons. 4. Search for 'pixel' or 'PixelSymphony' in the addon list and enable the addon by checking the box. 5. The addon will register custom panels in the Properties window under Collection properties and Object custom properties sections. 6. To use the system, select an object and navigate to the Object Properties > Custom Properties panel to add 'pix_' prefixed properties, or select a collection and use the PIXEL Collection Panel to mark collections. 7. Access rendering functions through the top menu bar under Render > Render Specific Frames (Modal) for automated frame processing.
