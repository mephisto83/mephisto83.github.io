# py-see-screen

![Language](https://img.shields.io/badge/language-Python-blue)

## Overview

py-see-screen is a Python-based screen analysis and image recognition service that provides automated detection of UI elements within desktop applications. The project implements a Flask REST API that can analyze screenshots to locate specific interface components, with specialized support for Discord and a posts application. The system uses OpenCV for computer vision tasks including template matching and QR code detection, enabling automated UI testing or screen automation workflows.

## Architecture

The application follows a modular architecture with a base DesktopComponent class that provides core image analysis functionality including template matching via cv2.matchTemplate and QR code detection using cv2.QRCodeDetector. Two specialized components inherit from this base: Discord class for Discord application analysis and Posts class for posts application analysis. The main index.py file serves as the Flask web server entry point, exposing a single POST endpoint at /api/v1/get that accepts current screen view data and returns coordinate information for detected UI elements. Platform-specific image templates are stored in organized resource folders (resources/discord/ and resources/posts/) with separate assets for macOS and other platforms.

## Key Features

- Template matching for UI element detection using OpenCV's cv2.matchTemplate with TM_SQDIFF_NORMED method
- Cross-platform support with platform-specific image assets for macOS and other systems
- Discord application analysis with detection of title bar, top-right and bottom-left UI components
- QR code detection and decoding capabilities using cv2.QRCodeDetector for multi-code scenarios
- REST API endpoint that accepts screenshot data and returns precise coordinate boundaries for detected elements
- Modular component system allowing easy extension for additional applications beyond Discord and posts

## Getting Started

1. Install Python dependencies: pip install flask flask-restful opencv-python pandas
2. Ensure the resources directory structure exists with subdirectories resources/discord/ and resources/posts/
3. Place platform-specific template images in appropriate resource folders (discord_title.png, discord_title_mac.png, bottom_left.png, bottom_left_mac.png, top_right.png, top_right_mac.png for Discord; qr_target.png for posts)
4. Run the Flask application: python index.py
5. The server will start on localhost:5123 with debug mode enabled
6. Send POST requests to http://localhost:5123/api/v1/get with JSON payload containing 'application' field (discord or posts) and 'current_view' field with path to screenshot image
7. The API will return JSON with coordinate data for detected UI elements (topRight, bottomLeft for Discord; target for posts)
