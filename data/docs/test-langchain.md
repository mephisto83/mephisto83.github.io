# test-langchain

![Language](https://img.shields.io/badge/language-Python-blue)

## Overview

test-langchain is a minimal Python tutorial project demonstrating text translation capabilities using Google's FLAN-T5-XXL transformer model. The repository serves as a learning exercise for implementing language model inference using Hugging Face's transformers library, specifically focusing on conditional text generation for translation tasks. Despite its name suggesting LangChain usage, the project actually utilizes direct transformer model implementation for English-to-German translation functionality.

## Architecture

The project follows a simple single-file architecture with index.py as the main entry point located in langchain/tutorial-env/. The application loads a pre-trained T5ForConditionalGeneration model and T5Tokenizer from Hugging Face's model hub, specifically the 'google/flan-t5-xxl' variant. The architecture implements a straightforward pipeline: text tokenization, GPU-accelerated model inference using CUDA device mapping, and output decoding. The pyvenv.cfg file indicates usage of a Python virtual environment for dependency isolation.

## Key Features

- English to German translation using FLAN-T5-XXL transformer model
- GPU acceleration with automatic device mapping via CUDA
- Hugging Face transformers integration for model loading and inference
- Token-based text processing with T5Tokenizer
- Pre-trained model inference without fine-tuning requirements

## Getting Started

1. Clone the repository and navigate to langchain/tutorial-env/ directory. 2. Create a Python virtual environment using the provided pyvenv.cfg configuration. 3. Install dependencies by running 'pip install -r requirements.txt' to install transformers==4.30.1, accelerate==0.20.3, and sentencepiece packages. 4. Ensure CUDA-compatible GPU is available and properly configured for PyTorch. 5. Execute the translation script with 'python index.py' - the script will automatically download the google/flan-t5-xxl model (several GB) on first run and perform English to German translation of the sample text 'How old are you?'.
