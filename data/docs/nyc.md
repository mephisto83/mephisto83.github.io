# nyc

![Language](https://img.shields.io/badge/language-JavaScript-blue)

## Overview

This project is a specialized web scraper and image downloader designed to collect map tiles from NYC.gov's GIS mapping service and images from AWS S3 storage. The repository contains both a TypeScript-based downloader application and browser-side JavaScript for collecting map metadata. The primary purpose is to systematically download and organize large collections of geographic map tiles and related images, creating local JSON indexes for easy access to the downloaded assets.

## Architecture

The application follows a simple Node.js architecture with TypeScript compilation. The main entry point is src/index.ts which contains multiple download functions: captureNewYorkData() for downloading NYC map tiles in a grid pattern across different zoom levels, captureIcanFeel() for downloading images from S3, and buildImageFiles() for generating JSON indexes. The src/dom.js file provides browser-side functionality to extract map tile information from the NYC GIS web interface. Downloaded images are organized in structured directories (image/backgrounds/, image/backgrounds-small/, image/imgs/) with corresponding JSON metadata files (backgroundImages.json, backgroundImages-small.json, images.json) that map file paths for web application usage.

## Key Features

- Automated NYC GIS map tile downloading with configurable zoom levels and grid coordinates
- S3 image batch downloading with error handling and timeout management
- Automatic directory structure creation and file organization
- JSON metadata file generation for downloaded image collections
- Browser-side map tile URL extraction from NYC GIS interface
- Hexadecimal coordinate formatting for NYC map tile URL construction
- Duplicate file detection to avoid re-downloading existing assets
- Robust error handling with timeout mechanisms for network requests

## Getting Started

1. Clone the repository and navigate to the project directory. 2. Install dependencies with 'npm install' to get TypeScript, rimraf, and Node.js type definitions. 3. Build the project using 'npm run build' which compiles TypeScript files from src/ to build/. 4. Run the application with 'npm start' or directly with 'node build/index.js'. 5. The application will automatically create the necessary directory structure (image/backgrounds/, image/backgrounds-small/, image/imgs/) and begin downloading images. 6. For browser-side map collection, load src/dom.js in the NYC GIS web interface to extract map tile URLs. 7. Monitor the console output for download progress and any failed requests. 8. Check generated JSON files (backgroundImages.json, backgroundImages-small.json, images.json) for organized file path references.
