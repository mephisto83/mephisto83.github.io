# chad-serve

![Language](https://img.shields.io/badge/language-TypeScript-blue)

## Overview

Chad-serve is a full-stack document search and indexing application that combines semantic search capabilities with a web-based interface. The system allows users to upload and index various document types (PDFs, text files, Excel, CSV, Jupyter notebooks) using FAISS (Facebook AI Similarity Search) vector embeddings, then perform intelligent queries against the indexed content. The application leverages sentence transformers and Hugging Face models for creating semantic embeddings, enabling contextual search rather than simple keyword matching.

## Architecture

The application follows a three-tier architecture with a Python Flask backend (pythonserver/), a React TypeScript frontend (web/chad-app/), and an endpoint testing utility (enpoint_tester/). The backend uses LangChain for document processing, FAISS for vector storage, and various document loaders for different file formats. The core functionality is implemented in lib.py with document processing functions, while functions.py handles the business logic including index management and search operations. The React frontend provides a dashboard interface with components for configuration management, search functionality, and progress monitoring. Communication between tiers occurs via REST APIs with CORS enabled.

## Key Features

- Multi-format document ingestion supporting PDFs, text files, Excel, CSV, and Jupyter notebooks via specialized loaders in lib.py
- Semantic search using FAISS vector indexing with configurable chunk sizes and embedding models
- Real-time indexing progress monitoring with SQLite-backed progress tracking in server.py
- Web-based configuration management allowing users to modify index settings, target folders, and model parameters
- Asynchronous document processing with threading support for non-blocking index creation
- PDF viewing and file opening capabilities directly from search results
- RESTful API endpoints for querying (/query), configuration management (/config, /update_config), and index operations (/ensure_index, /index_exists)

## Getting Started

1. Install Python 3 and Node.js as prerequisites. 2. Create a Python virtual environment using 'python -m venv my_venv'. 3. Activate the virtual environment (Windows: 'my_venv\Scripts\activate.bat', macOS/Linux: 'source my_venv/bin/activate'). 4. Install Python dependencies with 'pip install -r pythonserver/requirements.txt'. 5. Start the Flask backend server by running 'python pythonserver/server.py' (defaults to port 5000). 6. In a separate terminal, navigate to 'web/chad-app' and install frontend dependencies with 'npm install'. 7. Start the React development server with 'npm run start' (defaults to port 3000). 8. Access the web interface at http://localhost:3000 and configure the target folder path in the settings. 9. Use the ensure index functionality to process documents, then perform searches through the web interface.
