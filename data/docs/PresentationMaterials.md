# PresentationMaterials

> Presentation Materials

![Language](https://img.shields.io/badge/language-Python-blue)

## Overview

PresentationMaterials is a specialized multimedia content generation toolkit that combines Blender 3D rendering capabilities with PowerPoint presentation processing. The repository serves as a comprehensive pipeline for creating animated video presentations from 3D scenes, static images, and PowerPoint content. The system appears designed for automated presentation video generation, leveraging Blender's rendering engine to produce high-quality visual content with synchronized audio tracks.

## Architecture

The system follows a modular architecture with distinct components for different aspects of multimedia processing. The core rendering engine is built around Blender's Python API (bpy) with scripts like render.py handling 3D scene rendering and camera management. Video composition is managed through anim_video_editor.py and anim_video_editor_smart.py which use Blender's video sequence editor to combine rendered frames with audio tracks. The powerPointConverter.py module handles extraction of content from PowerPoint files, converting slides, shapes, charts, and text formatting into JSON structures. Asset management is handled through a comprehensive library of Blender files in the blend/ directory containing various 3D models, materials, and textures for architectural visualization including rooms, furniture, and surface materials with full PBR texture sets (diffuse, normal, displacement, specular maps).

## Key Features

- Automated video generation from image sequences using anim_video_editor.py with configurable frame counts and audio synchronization
- Advanced video composition through anim_video_editor_smart.py supporting JSON-based image mapping, custom animation settings, and multi-track audio synchronization
- PowerPoint to JSON conversion via powerPointConverter.py extracting slides, shapes, text formatting, charts, tables, and embedded images
- PNG metadata extraction through imageMeta.py for reading embedded text chunks and image properties
- 3D scene rendering pipeline with render.py supporting custom cameras, frame ranges, render borders, and ProSky environment integration
- Comprehensive 3D asset library including architectural elements (rooms, stairs, bleachers), PBR materials (brick, concrete, wood, tiles), and high-resolution texture sets
- PowerPoint generation from images using powerPointPrint.py for reverse workflow capabilities
- Configurable render settings including resolution, file formats, audio codecs, and transparency support

## Getting Started

Install Blender with Python scripting support and ensure the bpy module is available. Install python-pptx library using 'pip install python-pptx' for PowerPoint processing capabilities. Place 3D assets from the blend/ directory in the appropriate Blender library paths or reference them directly. Configure audio files in an 'audio' directory for video synchronization. Create an 'output' directory for rendered videos and images. To render videos, run 'blender -b -P anim_video_editor.py -- [image_path] [frame_count] [audio_path] [output_prefix]' with appropriate parameters. For PowerPoint conversion, execute 'python powerPointConverter.py [input.pptx] [output.json] [write_images] [image_output_dir]'. Configure JSON mapping files for advanced video composition using the smart editor. Ensure FFmpeg is properly configured in Blender for H.264 and MP3 encoding support.
