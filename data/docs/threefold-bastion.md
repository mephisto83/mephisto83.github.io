# threefold-bastion

![Language](https://img.shields.io/badge/language-TypeScript-blue)

## Overview

Threefold Bastion is a sophisticated 3D tower defense game built with React, TypeScript, and React Three Fiber. The game features a cyberpunk-themed military scenario where players defend their base against waves of enemies using various tower types, each with unique damage capabilities targeting either shields or hull health. The project demonstrates advanced 3D web game development with comprehensive internationalization support (English and French), rich multimedia integration including character voice acting and video introductions, and a complex progression system with multiple tech tiers and upgrade paths.

## Architecture

The application follows a component-based architecture with clear separation between 3D game logic and 2D UI overlays. The core game state is managed through Zustand stores in src/state/gameState.ts, while 3D components in src/game/components/ handle rendering of towers, enemies, and terrain using React Three Fiber. The game systems are modularized into src/game/systems/ for audio management, wave spawning, and core game logic. Asset loading is handled through src/game/utils/ with extensive public/ directory containing 3D models, audio files organized by character and language, sprite animations, and video assets. The UI layer in src/ui/ provides React-based overlays for HUD elements, menus, and game screens that interact with the 3D scene below.

## Key Features

- 3D tower defense gameplay with orbital camera controls and WebGL rendering
- Dual damage system with kinetic (gold projectiles) and energy (cyan projectiles) weapons targeting health vs shields
- Progressive tech tier system (Boot Sector, Precision Systems, Mobile Warfare, Industrial Control) with unlockable upgrades
- Comprehensive character system with named officers, multilingual voice acting, and video introductions
- Resource management economy with money and energy currencies, mining stations, and command node buffs
- Internationalization support for English and French with localized audio and subtitle files
- Rich multimedia integration including character catchphrases, background music, and sprite-based visual effects
- Firebase integration for cloud deployment and potential multiplayer features
- Modular tower types including Basic, Sniper, Cannon, Corsair towers plus support structures

## Getting Started

1. Clone the repository and ensure Git LFS is installed and configured to handle large media assets in public/ directory. 2. Run 'git lfs pull' to download all audio, video, and 3D model files. 3. Install dependencies with 'npm install' which will set up React, TypeScript, Vite, and all Three.js-related packages. 4. Start the development server with 'npm run dev' which launches Vite on port 5173. 5. Optionally run 'npm run generate-config' to regenerate character configuration files using scripts/generate_character_config.js. 6. For production deployment, use 'npm run build' which compiles TypeScript and bundles the application, then 'npm run deploy' which executes scripts/deploy_with_retry.sh for Firebase hosting.
