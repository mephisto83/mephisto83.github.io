# woodbury_instagram

![Language](https://img.shields.io/badge/language-JavaScript-blue)

## Overview

Woodbury Instagram is a Chrome extension that automates Instagram posting through web interface automation rather than using Instagram's official API. The extension provides a complete solution for posting images to Instagram from the browser, featuring a popup UI for image upload and caption entry, content script injection for DOM manipulation on instagram.com, and background service worker coordination. It integrates with the Woodbury ecosystem as an extension module, providing tools and commands for automated Instagram posting that can be used by AI agents or other automation workflows.

## Architecture

The extension follows Chrome extension manifest v3 architecture with three main components: background.js serves as the service worker handling message passing between components and tab management; content.js is injected into instagram.com pages to perform DOM automation using MutationObserver patterns and CSS selectors; popup.html/js provides the user interface for image selection and posting initiation. The system uses chrome.runtime.sendMessage for inter-component communication, implements comprehensive DOM selector strategies with fallbacks for Instagram's changing UI, and includes integration patterns for flow-frame-core desktop automation. State management is handled through pendingPosts Map in the background worker, with status updates propagated through the message system.

## Getting Started

1. Clone the repository and navigate to the woodbury_instagram folder. 2. Open Chrome and go to chrome://extensions/, then enable Developer mode toggle. 3. Click 'Load unpacked' and select the woodbury_instagram directory containing manifest.json. 4. Ensure you're logged into Instagram in a Chrome tab for the extension to detect the session. 5. For Woodbury ecosystem integration, install the Woodbury Bridge Chrome extension and ensure it's connected. 6. For desktop automation features, install flow-frame-core and configure file modal handling. 7. Click the extension icon in Chrome toolbar to access the popup interface, or use programmatically via the instagram_post tool in Woodbury-enabled environments. 8. Test posting by selecting an image through any supported method (upload, drag-drop, paste, URL) and clicking 'Post to Instagram'.
