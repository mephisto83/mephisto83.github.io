# suno-downloads

![Language](https://img.shields.io/badge/language-JavaScript-blue)

## Overview

suno-downloads is a Chrome extension-based tool designed to automate the extraction and downloading of audio files from Suno AI's music generation platform. The system combines a browser extension that identifies and extracts song metadata with download URLs, and a Python script that handles the actual file downloads and organization. The extension targets specific DOM elements on Suno's interface to locate song titles and extract corresponding MP3 URLs from React component props, while the Python downloader processes these URLs and organizes the files into a structured music directory.

## Architecture

The system follows a two-phase architecture: a client-side Chrome extension (background.js, content.js) that injects into Suno web pages to extract audio URLs, and a server-side Python script (download.py) that processes the extracted URLs. The extension uses Chrome's scripting API to inject content scripts that query DOM elements with specific selectors, extract React component data, and generate downloadable text files containing MP3 URLs. The Python component monitors the Downloads folder for files with the 'suno_download-' prefix, downloads the referenced MP3 files, and organizes them into artist/album folders while cleaning up the temporary URL files.

## Key Features

- Automated extraction of audio URLs from Suno AI React grid components using aria-label selectors
- Cross-platform Downloads folder detection supporting Windows, macOS, and Linux with XDG compliance
- Intelligent file organization with folder name cleaning that removes track numbers and special characters
- Duplicate file detection and skipping to prevent redundant downloads
- Error handling with selective cleanup - only removes source files after successful downloads
- Chrome extension integration with toolbar button activation and content script injection
- Local server communication capability for storing current song names via localhost:3000 endpoint

## Getting Started

1. Load the Chrome extension by navigating to chrome://extensions/, enabling Developer mode, and loading the 'chrome-extension/extension' folder as an unpacked extension. 2. Install Python dependencies with 'pip install requests' (pathlib is built-in). 3. Create a 'music' directory in your project root where downloaded files will be organized. 4. Navigate to a Suno AI page with generated songs and click the extension's toolbar icon to extract URLs and generate download files. 5. Run 'python download.py' to process any 'suno_download-*.txt' files in your Downloads folder and download the MP3s to organized subfolders. 6. Optional: Set up a local server on port 3000 if you want to use the setObject functionality for tracking current song names.
