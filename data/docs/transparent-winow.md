# transparent-winow

![Language](https://img.shields.io/badge/language-TypeScript-blue)

## Overview

This repository contains two related Electron/React applications focused on creating transparent window interfaces. The main project 'transparent-winow' is a standard React web application, while 'measurewin' is a more advanced Electron-based desktop application that creates transparent, frameless windows. The measurewin application appears to be designed for overlay functionality, potentially for screen measurement or annotation tools, leveraging Electron's ability to create transparent windows that can sit on top of other applications.

## Architecture

The repository follows a dual-project structure with two distinct applications. The 'transparent-winow' directory contains a standard Create React App setup using JavaScript, while 'measurewin' implements a more sophisticated Electron application with TypeScript. The measurewin architecture consists of an Electron main process (main.ts) that creates transparent, frameless BrowserWindows, and a React renderer process for the UI. The main process handles window creation with transparency enabled and frame disabled, while the React frontend provides the user interface. Hot reloading is configured for development, and the build process compiles TypeScript to JavaScript before packaging with electron-builder.

## Getting Started

For the transparent-winow project: 1) Navigate to the transparent-winow directory, 2) Run 'npm install' to install dependencies, 3) Execute 'npm start' to launch the development server on localhost:3000, 4) Run 'npm test' for testing and 'npm run build' for production builds. For the measurewin Electron app: 1) Navigate to the measurewin directory, 2) Run 'npm install' or 'yarn install' to install all dependencies including Electron, 3) Use 'yarn electron:dev' to start the development environment which concurrently runs the React dev server on port 3541, compiles TypeScript, and launches Electron, 4) For production builds, execute 'yarn electron:build' which builds React, compiles TypeScript, and packages the Electron application using electron-builder. Ensure Node.js and npm/yarn are installed before beginning setup.
