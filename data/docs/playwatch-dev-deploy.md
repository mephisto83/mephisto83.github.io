## Overview

This is a Firebase-based backend deployment system for PlayWatch, a media streaming and content management platform. The repository serves as the development deployment environment, handling user authentication, media content management, player administration, and comprehensive metrics collection. The system is built around a microservices architecture with auto-generated TypeScript controllers and services for each data model, providing RESTful APIs for frontend applications and administrative interfaces.

## Architecture

The architecture follows a Firebase Cloud Functions pattern with Express.js routing, utilizing Firestore as the primary database. The codebase is heavily auto-generated with consistent patterns across all entities - each model (metricDeployment, socialMediaAccount, etc.) has corresponding service files with CRUD operations, property definitions, and model interfaces. The system implements a denormalized data structure with meta-properties for efficient querying, version control for all entities, and update propagation mechanisms. Key architectural components include: functions/src/controllers/auto/ containing auto-generated REST endpoints, functions/src/functions/ with service layer implementations, functions/src/models/ defining TypeScript interfaces and business logic, and a centralized firestore.ts for database connections.

- Auto-generated CRUD operations for 30+ data models including media, players, customers, and metrics
- Comprehensive metrics deployment system with configurable metric configurations and real-time monitoring
- Social media account management with link tracking and type categorization
- Player role management with invitation system for content creators
- Content house and media resource management with approval workflows
- Device management and customer administration with settings persistence
- Version control and soft deletion for all entities with created/updated timestamps
- Denormalized data structure with meta-properties for efficient Firestore queries
- Integration with Stripe for payment processing and Datadog for monitoring

## Getting Started

1. Install dependencies: cd functions && npm install 2. Set up Firebase project and configure firebase.json with your project ID 3. Place Firebase service account key as playwatch-dev-test-793123b161be.json in functions directory 4. Build TypeScript: npm run build 5. Start Firebase emulators for local development: npm run serve-all 6. Deploy functions only: npm run deploy 7. Deploy hosting only: npm run publish 8. Deploy everything: npm run all. The system requires Node.js 16 and uses cross-env for environment variable management. For testing, run npm test using the Mocha framework.
