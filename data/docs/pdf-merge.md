# pdf-merge

![Language](https://img.shields.io/badge/language-Python-blue)

## Overview

pdf-merge is a Python utility for combining multiple PDF files from a specified directory into a single merged PDF document. The tool automatically generates a table of contents with bookmarks for each merged file, making it easy to navigate through the combined document. It's particularly useful for consolidating collections of related PDF documents, such as study materials or documentation sets, into a single organized file.

## Architecture

The application follows a simple procedural architecture with two main functions in index.py. The combine_pdfs() function handles the core PDF merging logic using PyPDF2's PdfMerger class, iterating through PDF files in a directory and adding them to the merger with corresponding outline bookmarks. The main() function serves as the entry point with hardcoded directory and output file paths. The architecture leverages PyPDF2's built-in capabilities for PDF manipulation, file reading, and bookmark generation.

## Tech Stack

- Python
- PyPDF2

## Key Features

- Batch PDF merging from a specified directory
- Automatic table of contents generation with bookmarks for each merged file
- Preserves original file names as bookmark titles
- Sequential page numbering across all merged documents
- Simple single-file execution with configurable input directory and output filename

## Getting Started

1. Install Python 3.x on your system. 2. Install the PyPDF2 dependency using 'pip install PyPDF2'. 3. Clone or download the repository containing index.py. 4. Modify the 'directory' variable in the main() function to point to your PDF source folder (currently set to 'F:\pdfs\finra\series 6'). 5. Optionally modify the 'output_file' variable to change the output filename (default: 'combined.pdf'). 6. Run the script using 'python index.py' from the command line. The merged PDF will be created in the same directory as the script.
