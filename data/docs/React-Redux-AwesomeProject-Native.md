## Overview

React-Redux-AwesomeProject-Native is a cross-platform mobile application built with React Native that demonstrates a job management system with Redux state management. The application appears to be a prototype for managing music-related jobs or tasks, specifically dealing with MIDI files and blended music processing. The app provides functionality to view, create, edit, and delete jobs while displaying detailed logs and status information. The project serves as a comprehensive example of integrating React Native with Redux for state management, featuring both iOS and Android platform support with a RESTful API backend integration.

## Architecture

The application follows a standard Redux architecture pattern with clearly separated concerns. The state management is centralized through Redux with actions (jobAction.js), reducers (jobReducer.js, projectReducer.js), and a configured store (configureStore.js). The UI layer consists of presentational components (Job.js, JobList.js, Details.js, EditJob.js) and container components (App.js, BlendedList.js) that connect to the Redux store via react-redux. The application uses a RESTful API (blendedmusic.azurewebsites.net) for backend communication, handling CRUD operations for jobs and retrieving log details. Platform-specific entry points (index.android.js, index.ios.js) initialize the appropriate native components for each platform.

## Getting Started

1. Install Node.js and React Native CLI globally. 2. Clone the repository and navigate to the project directory. 3. Run 'npm install' to install dependencies including react-native, redux, react-redux, redux-thunk, react-native-vector-icons, and react-native-action-button. 4. For Android: Ensure Android SDK is installed and configured, then run 'react-native run-android' or use Android Studio to build the project from the android/ directory. 5. For iOS: Ensure Xcode is installed, navigate to ios/ directory, open AwesomeProject.xcodeproj in Xcode, and build/run the project or use 'react-native run-ios'. 6. Start the React Native packager with 'npm start' or 'react-native start'. 7. Configure the API endpoint in actions/jobAction.js if needed (currently points to blendedmusic.azurewebsites.net). 8. The app will initialize and fetch jobs from the configured API endpoint automatically on first launch.
