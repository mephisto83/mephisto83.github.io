# alkmene

> measure the distance of an object moved.

![Language](https://img.shields.io/badge/language-TypeScript-blue)

## Overview

Alkmene is a React-based web application that leverages TensorFlow.js to measure and predict the distance of object movement through machine learning. The application allows users to interactively select points on images to create vectors, then uses neural network models to predict distances based on coordinate data. Built as a proof-of-concept for computer vision applications, it demonstrates how browser-based machine learning can be applied to spatial measurement tasks without requiring server-side processing.

## Architecture

The application follows a standard React architecture with TypeScript, centered around the main App.tsx component that orchestrates the user interface. The core functionality is split between ImagePointCollection.tsx which handles user interaction for point selection and vector creation, and Prediction.ts which contains the TensorFlow.js machine learning logic. The ML pipeline uses linear regression and multi-layer perceptron models to learn from vector data (point coordinates and distances), with data normalization handled through custom tensor processing functions. Static image assets are served from the public directory, and the application includes a large dataset (data.json) for training purposes.

## Key Features

- Interactive point selection on images for vector creation through ImagePointCollection.tsx component
- Machine learning-based distance prediction using three model types: linear regression, single-hidden-layer MLP, and dual-hidden-layer MLP in Prediction.ts
- Real-time tensor data normalization with coordinate and distance scaling for optimal model training
- Browser-based neural network training with configurable epochs (1200) and learning rate (0.01) parameters
- Visual feedback system with Material-UI components for user interaction and results display
- Extensible model architecture supporting different neural network configurations through modular prediction functions

## Getting Started

1. Clone the repository and navigate to the alkmene directory. 2. Install dependencies using npm install or yarn install (both package-lock.json and yarn.lock are provided). 3. Start the development server with npm start or yarn start, which will launch the app at http://localhost:3000. 4. The application will automatically load with the default React development configuration. 5. For production deployment, run npm run build to create an optimized build in the build folder. 6. Ensure the data/data.json file is accessible for the machine learning training data. 7. The app requires a modern browser with WebGL support for TensorFlow.js functionality.
