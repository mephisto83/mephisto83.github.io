# wave-onset-detector

![Language](https://img.shields.io/badge/language-Python-blue)

## Overview

Wave-onset-detector is a Python-based audio analysis toolkit designed to detect and analyze musical events in WAV audio files. The project provides functionality to identify onset and offset timing of musical notes, convert audio frequencies to MIDI note numbers, and generate structured data output for further music processing applications. The system is particularly suited for processing stems or isolated audio tracks, as evidenced by the directory structures targeting DJ software exports like Serato stems.

## Architecture

The architecture consists of three independent Python modules that can be executed sequentially or separately. check.py serves as a basic onset detection processor using the aubio library to identify note attack points. detect_onsets.py extends this functionality by adding offset detection through RMS-based silence analysis. detect_midi_note.py performs frequency analysis using FFT to determine pitch content and convert it to MIDI note numbers. All modules follow a batch processing pattern, iterating through directories of WAV files and outputting JSON data files with corresponding names.

## Key Features

- Onset detection using aubio's default onset detection algorithm with 512-sample FFT windows
- Offset detection through RMS amplitude analysis with configurable silence thresholds and minimum duration parameters
- Frequency-to-MIDI conversion using A4=440Hz reference with logarithmic pitch calculation
- Batch processing of entire directories containing WAV audio files
- JSON output format preserving timing information in seconds and MIDI note numbers
- Support for both mono and stereo audio files with automatic channel reduction
- Configurable analysis parameters including window size, hop size, and silence detection thresholds

## Getting Started

1. Install required Python dependencies: pip install aubio numpy scipy
2. Update the directory paths in each script (check.py, detect_onsets.py, detect_midi_note.py) to point to your WAV file locations
3. For basic onset detection: run python check.py to generate JSON files with onset timestamps
4. For onset/offset detection: run python detect_onsets.py to generate JSON files with both onset and offset timing
5. For MIDI note detection: ensure JSON files from detect_onsets.py exist, then run python detect_midi_note.py to generate track_data.json with MIDI note information
6. Output JSON files will be created in the same directory as the input WAV files
