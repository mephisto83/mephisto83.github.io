# isometric-html-test

![Language](https://img.shields.io/badge/language-TypeScript-blue)

## Overview

isometric-html-test is a React-based web application that demonstrates isometric visualization techniques using HTML, CSS, and TypeScript. The project implements a sophisticated isometric city builder or map visualization system that combines Voronoi diagram generation with procedural city asset placement. The application features interactive isometric rendering of city elements like apartments and stores, with procedural road generation and spatial layout algorithms. This appears to be a proof-of-concept or educational project exploring computational geometry, procedural generation, and isometric game development techniques in a web browser environment.

## Architecture

The application follows a React component-based architecture with specialized modules for geometric computations and rendering. The core architecture consists of: (1) React components in src/ for UI and rendering (App.tsx, IsoComponent.tsx, IsometricMenu.tsx), (2) A complete Voronoi diagram implementation in src/voronoi/ with classes for computational geometry (Voronoi.ts, Cell.ts, Edge.ts, Vertex.ts), (3) Asset management through NamedCityAsset.tsx for handling isometric building sprites, (4) Utility functions in util.ts for seeded random number generation and spatial algorithms, and (5) Styling through SCSS files for isometric visual effects. The application generates procedural city layouts using Voronoi tessellation, places building assets at calculated positions, and renders everything in an isometric perspective using CSS transforms and absolute positioning.

## Key Features

- Procedural Voronoi diagram generation for city layout with customizable cell count and seeded randomization
- Isometric asset rendering system supporting multiple building types (apartments, stores) with precise positioning data
- Custom Voronoi tessellation implementation with complete geometric primitives (vertices, edges, cells, halfedges)
- Seeded pseudo-random number generation using cyrb128 hash and sfc32 algorithms for reproducible city generation
- Interactive isometric menu system with SCSS-based styling for 3D visual effects
- Road connection system with SVG-based rendering for connecting city districts
- Geometric utility functions including convex hull calculation and point-in-polygon testing
- Circle-line intersection algorithms for spatial collision detection and asset placement

## Getting Started

1. Clone the repository and navigate to the my-app directory. 2. Install dependencies by running 'npm install' which will install React 18.2.0, TypeScript 4.8.4, SASS, and other required packages. 3. Start the development server with 'npm start' to run the application on http://localhost:3000. 4. The application will load with an isometric city visualization featuring procedurally generated Voronoi cells and placed building assets. 5. For testing, run 'npm test' to execute the Jest test suite. 6. To build for production, use 'npm run build' which creates an optimized build in the build folder. 7. The main application logic is in src/App.tsx and isometric components can be found in src/IsoComponent.tsx and src/IsometricMenu.tsx.
