# BubbleTest

![Language](https://img.shields.io/badge/language-JavaScript-blue)

## Overview

BubbleTest is a lightweight JavaScript-based comic book bubble generator that creates customizable speech and thought bubbles on HTML5 canvas. The project provides a simple yet flexible API for drawing comic-style bubbles with rounded corners, directional pointers, and automatic text wrapping capabilities. It serves as a utility tool for web developers, comic creators, or educational applications that need to dynamically generate comic book-style dialogue bubbles with varying pointer positions and orientations.

## Architecture

The application follows a simple client-side architecture with a single HTML file (index.html) serving as the entry point, containing a canvas element for rendering. The core functionality resides in comicBubbles.js, which implements the bubble drawing logic using HTML5 Canvas 2D context. The drawBubble function serves as the main API, accepting parameters for position, dimensions, text content, and pointer configuration. A supporting wrapText utility function handles automatic text wrapping within bubble boundaries. The project uses http-server for local development serving, with no backend dependencies or complex build processes.

## Key Features

- Dynamic comic bubble generation with customizable dimensions and positioning
- Four-directional pointer support (top, bottom, left, right) with configurable positioning
- Automatic text wrapping with overflow handling using Comic Sans MS font styling
- Rounded corner bubble design with quadratic curve implementation
- Configurable pointer geometry including width, distance, and base position offsets
- Canvas-based rendering for high-quality graphics and potential export capabilities
- Zero-dependency runtime execution requiring only a modern web browser

## Getting Started

1. Clone the repository from https://github.com/mephisto83/BubbleTest. 2. Navigate to the project directory and run 'npm install' to install the http-server development dependency. 3. Start the development server using 'npm start' which launches http-server. 4. Open your web browser and navigate to the local server address (typically http://localhost:8080). 5. The application will load index.html automatically, executing comicBubbles.js to render example bubbles on the canvas. 6. Modify the drawBubble function calls in comicBubbles.js to customize bubble appearance, text content, and pointer configurations. 7. Refresh the browser to see changes reflected immediately.
