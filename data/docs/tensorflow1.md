## Overview

This project is a Node.js-based automation toolkit for TensorFlow object detection model training and configuration. It provides JavaScript utilities to streamline the setup and execution of TensorFlow Object Detection API workflows, specifically focused on training models with PASCAL VOC and MS COCO datasets. The toolkit serves as a wrapper around TensorFlow's Python-based object detection pipeline, offering configuration templating and command generation for model training workflows.

## Architecture

The project follows a modular architecture with two main Node.js modules: process.js handles configuration file generation from templates and dataset preparation, while object_detection.js manages the execution of TensorFlow training commands. The system uses template-based configuration where configuration.template files in src/faster_rcnn_inception/ are processed with dynamic values to generate TensorFlow pipeline config files. The architecture supports file operations through shelljs for cross-platform compatibility and uses child_process.spawn for executing Python TensorFlow commands. Configuration files (pascal_config.json, pet_config.json) drive the parameterization of the entire training pipeline.

## Getting Started

1. Create Anaconda environment: conda create -n tensorflow1 pip python=3.5 2. Activate environment: activate tensorflow1 3. Set Python path: set PYTHONPATH=C:\tensorflow1\models;C:\tensorflow1\models\research;C:\tensorflow1\models\research\slim 4. Install TensorFlow: pip install --ignore-installed --upgrade tensorflow-gpu (or tensorflow for CPU) 5. Install dependencies: conda install -c anaconda protobuf && pip install pillow lxml Cython jupyter matplotlib pandas opencv-python 6. Navigate to models/research and compile protobuf files using the provided protoc command 7. Build and install TensorFlow Object Detection API: python setup.py build && python setup.py install 8. Verify installation: python object_detection/builders/model_builder_test.py 9. Configure dataset paths in pascal_config.json or pet_config.json 10. Run configuration generation: npm run pascal_config 11. Execute training: npm run object_detection_pascal 12. Monitor training with TensorBoard: tensorboard --logdir={model_directory}
