# rbanq-mobile

![Language](https://img.shields.io/badge/language-TypeScript-blue)

## Overview

rbanq-mobile is a React Native mobile banking simulation application that combines traditional banking features with an innovative city-building game mechanic. The app allows users to manage virtual banks, create and trade digital assets, and participate in an economic simulation where banking operations are visualized through an isometric city interface. The application leverages Firebase for backend services and implements a sophisticated Redux-based state management system to handle complex financial transactions, user authentication, and real-time city visualization.

## Architecture

The application follows a feature-based modular architecture with clear separation of concerns. The core structure is built around Redux slices for state management (bankCity-slice.ts, cityAsset-slice.ts), with each feature containing its own model definitions, property definitions, validation rules, services, and thunks. The city visualization system is implemented through a custom component hierarchy in components/digicity/ using Voronoi diagram generation for procedural city layouts. Firebase integration handles user authentication and data persistence with custom security rules. The UI components are built with React Native and utilize a themed component system for consistent styling across the application.

- Virtual banking system with account management, transactions, and investment tracking
- Digital asset creation and management with real-time market simulation
- Isometric city visualization with interactive building placement and management
- User authentication and authorization through Firebase with role-based access controls
- Real-time financial data visualization including profit graphs and business plan evaluations
- QR code integration for mobile banking operations
- Auction system for trading digital assets and business shares
- Mathematical expression rendering for complex financial calculations
- Offline-first architecture with Firebase synchronization

## Getting Started

1. Clone the repository and navigate to the rbanq-mobile directory. 2. Install dependencies using npm install or yarn install. 3. Configure Firebase by adding your firebase configuration to components/FirebaseContext.tsx. 4. Set up Firebase Firestore security rules using the provided .rules files in each feature directory (bankCity-rules.rules, cityAsset-rules.rules). 5. Configure the Expo development environment by running expo install. 6. Install required fonts by placing Bangers-Regular.ttf and SpaceMono-Regular.ttf in the assets/fonts/ directory. 7. Set up platform-specific configurations in app.json for both iOS and Android builds. 8. Run the development server using expo start or npm start. 9. For production builds, configure the appropriate signing certificates and build using expo build:android or expo build:ios. 10. Ensure all required permissions for camera access (QR code scanning) are properly configured in the platform-specific configuration files.
