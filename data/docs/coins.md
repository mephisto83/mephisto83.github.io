# coins

![Language](https://img.shields.io/badge/language-JavaScript-blue)

## Overview

This is a React Native cryptocurrency tracking and trading application called 'coins' that provides real-time market data visualization, portfolio management, and automated trading capabilities. The app integrates with multiple cryptocurrency exchanges including CoinMarketCap, Bittrex, and Binance to fetch market data, display charts, and execute trading strategies. The project includes both a mobile frontend built with React Native and a Node.js backend bot system that implements genetic algorithms and reinforcement learning for automated trading analysis.

## Architecture

The application follows a Redux-based architecture with the main React Native app in the coins/ directory and a separate Node.js bot system in coins/bot/. The frontend consists of multiple screens (RouteScreen, ConfigureCoins, Top24, Graphs, etc.) managed by react-native-router-flux, with state management handled through Redux actions in coins/actions/coinactions.js. The backend bot system includes genetic algorithm implementations (genetic.js), reinforcement learning modules (rl.js), trading world simulations (world.js), and cryptocurrency data services (coinservice.js). Data visualization is handled through custom chart components in the components/ directory, utilizing react-native-pathjs-charts for candlestick and market data visualization.

## Getting Started

1. Clone the repository and navigate to the coins/ directory. 2. Install dependencies using 'npm install' to get React Native, Redux, and other required packages. 3. For Android development, ensure Android SDK is installed and configure android/local.properties. 4. Run 'npm run android' to build and run the React Native app on Android device/emulator. 5. For the bot system, run 'npm run build:node' to compile the bot code from coins/bot/ to coins/bot_dist/ using Babel. 6. Execute the trading bot with 'npm run run:node' to start the genetic algorithm and reinforcement learning modules. 7. Configure API keys for cryptocurrency exchanges in the service files if live trading is required. 8. Use 'npm test' to run Jest tests for the application components.
