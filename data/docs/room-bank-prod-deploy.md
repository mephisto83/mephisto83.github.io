## Overview

Room Bank is a comprehensive Firebase-based educational banking simulation platform that enables virtual economic systems for classrooms. The system provides a complex ecosystem where students can participate in virtual banking, business operations, city development, and economic transactions through a gamified learning environment. The platform supports multiple educational entities including schools, districts, classrooms, and individual student accounts, with integrated payment processing through Stripe for real-world monetization of educational services.

## Architecture

The application follows a serverless architecture built on Google Firebase with Cloud Functions handling business logic and Firestore for data persistence. The codebase is structured around auto-generated CRUD controllers for entities like banks, schools, districts, professors, and students, with a sophisticated model system supporting relationships between entities. Key architectural components include: functions/src/controllers/auto/ containing entity-specific controllers, functions/src/functions/ with business logic services, and functions/src/models/ defining data models and relationships. The system uses a snapshot-based update mechanism for maintaining data consistency across related entities, with automated relationship management through On_*_Update and Perform_Updates_For_* functions.

- Comprehensive auction system for virtual assets and business opportunities with automated generation capabilities
- Integrated payment processing via Stripe for course payments and premium features
- Real-time transaction processing and account management with detailed audit logging
- Role-based permission system supporting directors, professors, students, and administrators
- Automated job scheduling for account cleanup, auction generation, and employee management
- Digital asset management system for educational resources and virtual business assets
- Comprehensive analytics and reporting for educational outcomes and system usage

## Getting Started

1. Install Node.js 16 and Firebase CLI globally. 2. Clone the repository and navigate to the project root. 3. Run 'npm install' in the root directory and 'cd functions && npm install' for function dependencies. 4. Configure Firebase project by updating firebase.json and firestore configuration files. 5. Set up Google Cloud service account credentials by placing the admin SDK key file (referenced in package.json serve-all script) in the functions directory. 6. Configure Stripe API keys for payment processing. 7. Deploy Firestore rules using 'firebase deploy --only firestore:rules'. 8. Build TypeScript code with 'npm run build'. 9. Deploy functions using 'npm run deploy' or deploy everything with 'npm run all'. 10. For local development, use 'npm run serve-all' to start Firebase emulators with proper credentials.
