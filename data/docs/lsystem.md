## Overview

This is a lightweight L-system (Lindenmayer system) implementation written in JavaScript for generating fractal patterns and recursive structures. L-systems are parallel rewriting systems used to model the growth processes of plants, create fractal art, and simulate biological development patterns. The project provides a simple rule-based engine where users can define initial conditions and transformation rules that are applied iteratively to generate complex patterns from simple starting points.

## Architecture

The system follows a functional closure pattern with the main lsystem.js module exposing a factory function that creates isolated L-system instances. Each system maintains internal state including the current pattern array, initial conditions, and a collection of transformation rules. The core execution engine in the execute() method processes arrays recursively, applying matching rules to transform elements. The architecture supports nested array structures and rule-based transformations through a match/outFunc pattern where rules define both matching conditions and output generation logic.

## Key Features

- Rule-based pattern generation with configurable match and output functions
- Recursive execution engine that handles nested array structures
- Step-by-step iteration through the step() method for controlled pattern evolution
- Flexible rule system supporting custom transformation logic
- Browser-ready implementation with global window.lsystem exposure
- State management with current pattern tracking and rule collection
- Debug support through print() method for pattern inspection

## Getting Started

1. Clone the repository and open index.html in a web browser. 2. The lsystem.js module is automatically loaded and exposes window.lsystem globally. 3. Create a new L-system instance by calling window.lsystem(initialValue) with your starting pattern. 4. Add transformation rules using addRule(matchFunction, outputFunction) where matchFunction determines if a rule applies and outputFunction defines the transformation. 5. Initialize the system with init() to set the initial state. 6. Execute pattern generation using step() for single iterations or execute() for immediate transformation. 7. Use print() to inspect the current pattern state in the browser console.
