# ComposerCompanion

![Language](https://img.shields.io/badge/language-JavaScript-blue) ![Docker](https://img.shields.io/badge/Docker-ready-blue)

## Overview

ComposerCompanion is a comprehensive web-based music composition and notation application built with React. The project serves as a digital music creation suite that allows users to compose, edit, visualize, and playback musical compositions through an intuitive interface. It leverages machine learning capabilities through TensorFlow.js and Magenta Music for intelligent music generation and assistance, while providing traditional music notation rendering through VexFlow and MIDI functionality for audio playback and export.

## Architecture

The application follows a modern React-based architecture with Redux Toolkit for state management, deployed on Firebase/Google Cloud Platform. The main application resides in the composer-companion-app directory, featuring a component-based UI architecture with extensive use of music-specific libraries. The codebase includes multiple sub-applications (cc-app as a simpler React app), code generation utilities for Redux slices and models, and comprehensive music notation rendering through integrated VexFlow and score-library components. The backend integration appears to handle MIDI processing, Firebase data persistence, and real-time music collaboration features.

## Getting Started

1. Clone the repository and navigate to composer-companion-app directory
2. Install dependencies: npm install
3. Configure Firebase by setting up firebase.json with your project credentials
4. Generate instrument data: npm run instruments
5. Compile SCSS styles: npm run style
6. Start development server: npm start (automatically runs version update script)
7. For production build: npm run build
8. Optional: Set up Firebase hosting and deploy using firebase deploy
9. For the simpler cc-app version, navigate to cc-app directory and run yarn install followed by yarn start
10. Ensure all required fonts and music notation assets are properly loaded from public/Data directory
