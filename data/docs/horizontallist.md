# horizontallist

![Language](https://img.shields.io/badge/language-JavaScript-blue)

## Overview

HorizontalList is a React-based interactive timeline component that demonstrates dynamic horizontal list rendering with flexible item widths. The project creates a visual timeline interface where users can add items with randomly generated widths and durations, then dynamically adjust the selected item's width using a range slider control. Built as a proof-of-concept for timeline visualization, it showcases React component state management, Redux integration, and cross-browser compatibility with Internet Explorer detection and polyfills.

## Architecture

The application follows a standard React-Redux architecture with clear separation of concerns. The main entry point (index.js) imports the app initialization from src/app.js, which sets up the Redux store with thunk middleware and renders the Provider-wrapped App component. The core components include App (src/components/app.js) as the main container managing timeline data and item selection, and Timeline (src/components/timeline.js) as the horizontal list renderer. State management is handled through Redux with a minimal reducer structure (src/reducers/reducer.js) combined in src/reducers.js. Cross-browser compatibility is ensured through polyfills in src/poly.js, including Object.assign polyfill and Internet Explorer detection utilities.

## Key Features

- Dynamic timeline item generation with random widths and timestamps
- Interactive item selection with visual feedback through click handlers
- Real-time width adjustment using HTML5 range slider controls
- Horizontal scrollable timeline layout using CSS flexbox
- Cross-browser Internet Explorer compatibility with polyfills and fallback styles
- Redux state management for scalable component communication
- Hot reloading development setup with Watchify and Babel transpilation

## Getting Started

1. Clone the repository and navigate to the project directory. 2. Install dependencies using 'npm install' to get all required packages including React, Redux, and Babel tooling. 3. Run 'npm run watchify' to start the development server with automatic transpilation and bundling - this watches index.js and outputs to bundle.js. 4. Open index.html in a web browser to view the application. 5. Use the 'Add' button to create new timeline items and click on any blue timeline item to select it. 6. Adjust the selected item's width using the range slider above the timeline. 7. For production builds, the bundle.js file contains the compiled application ready for deployment.
