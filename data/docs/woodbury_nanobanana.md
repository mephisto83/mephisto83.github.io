# woodbury_nanobanana

![Language](https://img.shields.io/badge/language-JavaScript-blue)

## Overview

Nano Banana is a Chrome browser extension that integrates Google's Gemini AI image generation capabilities directly into the browser. The extension provides both text-to-image generation and image editing functionality using Google's gemini-2.5-flash-image and gemini-3-pro-image-preview models. It serves as both a standalone Chrome extension and a Woodbury framework extension, offering a user-friendly interface for AI-powered image creation with features like generation history, multiple aspect ratios, and various quality settings.

## Architecture

The extension follows Chrome Extension Manifest V3 architecture with a background service worker (background.js) handling API communications, a popup interface (popup.html/js) for user interactions, and an options page (options.html/js) for configuration. The core functionality is split between browser-based Chrome extension components and a Node.js module (index.js) for Woodbury framework integration. The background.js acts as the API layer, making requests to Google's Generative Language API, while popup.js manages the UI state and user interactions. Settings and generation history are stored using Chrome's storage APIs.

## Key Features

- Text-to-image generation using Google Gemini models with detailed prompt support
- Image editing functionality allowing users to upload images and describe modifications
- Dual model support: gemini-2.5-flash-image for fast generation and gemini-3-pro-image-preview for high-quality 4K output
- Multiple aspect ratio options (1:1, 2:3, 3:2, 3:4, 4:3, 4:5, 5:4, 9:16, 16:9, 21:9) and image sizes up to 4K
- Generation history with local storage for viewing and reusing previous creations
- Drag-and-drop image upload interface with preview functionality
- API key management with connection testing and secure local storage
- Woodbury framework integration as a tool providing programmatic access to image generation
- Download and copy functionality for generated images with base64 data handling

## Getting Started

1. Clone the repository and navigate to the woodbury_nanobanana directory. 2. Convert SVG icons to PNG format using Inkscape, ImageMagick, or online converters (inkscape icon16.svg -w 16 -h 16 -o icon16.png for each size). 3. Open Chrome and navigate to chrome://extensions/. 4. Enable Developer mode using the toggle in the top right. 5. Click 'Load unpacked' and select the woodbury_nanobanana folder. 6. Obtain a Gemini API key from Google AI Studio (https://aistudio.google.com/app/apikey). 7. Click the extension icon in Chrome, navigate to Settings, and paste your API key. 8. Click 'Save Settings' and 'Test Connection' to verify. For Woodbury framework integration, ensure GEMINI_API_KEY is set in the extension's environment variables and the extension will register the 'nanobanana' tool automatically.
