## Overview

Redmath is a comprehensive JavaScript mathematical computation library that provides advanced mathematical operations including symbolic expression evaluation, calculus operations, linear algebra, and machine learning tensor operations. The library implements a sophisticated expression evaluation system with support for symbolic mathematics, derivatives, integrals, trigonometric functions, and matrix operations. It appears to be designed for both browser and Node.js environments with ES6 modules and Babel transpilation, targeting mathematical computation needs in web applications and educational software.

## Architecture

The architecture follows a modular design pattern with core mathematical components organized under redmath/source/math/. The Expression.js class serves as the foundation for all mathematical expressions, with specialized evaluators in expression/Evaluator.js handling symbolic computation. The Factor.js module provides factorization utilities, while ExpressionMatch.js implements pattern matching for mathematical transformations. Arithmetic operations are implemented as separate modules (Addition.js, Division.js, Multiplication.js, Subtraction.js) extending the base Expression class. The library includes specialized modules for calculus (Calculus.js), linear algebra (Matrix.js, Vector.js, Quaternion.js), 3D mathematics (J3DIMath.js, Matrix3d.js), signal processing (FFT.js), and machine learning (ml/tensor.js with TensorFlow.js integration). The build system uses Babel for ES5 transpilation and supports both development and production builds.

## Key Features

- Symbolic mathematical expression evaluation and manipulation through Expression.js and Evaluator.js
- Comprehensive calculus operations including derivatives and integrals with symbolic computation
- Linear algebra operations with Matrix.js, Vector.js, and Quaternion.js supporting 2D/3D transformations
- Trigonometric and hyperbolic function evaluation with support for sin, cos, tan, sec, csc, cot and their hyperbolic variants
- Advanced mathematical utilities including Fast Fourier Transform (FFT.js) and Hamilton operations
- Machine learning tensor operations integrated with TensorFlow.js in ml/tensor.js
- 3D mathematics support through J3DIMath.js and Matrix3d.js for graphics applications
- Pattern matching system via ExpressionMatch.js for mathematical expression transformations
- Factorization utilities in Factor.js for algebraic manipulation
- MathJax integration for mathematical rendering and display

## Getting Started

1. Clone the repository and navigate to the redmath directory. 2. Install dependencies using 'npm install' which will install Babel, TensorFlow.js, Mocha, and other required packages. 3. Build the project using 'npm run build' to transpile ES6 code to ES5 in the distribution directory. 4. For development, use 'npm run watchify' to automatically rebuild on file changes. 5. Run tests using 'npm test' which executes Mocha tests on .spec.js files with Babel compilation. 6. For tensor operations, run 'npm run tensor' to execute the TensorFlow.js tensor examples. 7. To start the Express.js server, use 'npm run nodeserver' which builds and runs experiments/nodeserver.js. 8. The main entry point is distribution/bundle.js after building, or import modules directly from source/ for ES6 environments.
