# a-frame-components

> Components for a-frame

![Language](https://img.shields.io/badge/language-TypeScript-blue)

## Overview

The a-frame-components repository is a comprehensive TypeScript-based library that extends A-Frame's capabilities with custom GUI and interactive components for virtual reality web applications. It provides a collection of reusable UI components specifically designed for 3D/VR environments, including buttons, containers, menus, sliders, and various interactive elements. The library serves as a bridge between React applications and A-Frame scenes, offering both declarative component usage and programmatic event handling utilities.

## Architecture

The architecture is organized into two main modules: gui components (src/gui/) and React utilities (src/react/). The gui module contains A-Frame component definitions that register custom elements like 'frame-container', 'frame-button', and 'base-interactive', each with their own schemas, lifecycle methods, and 3D rendering logic. The React module provides hooks and utilities for event management (useEventListenerOn, useBindEventOn), canvas manipulations, and various brush tools for 3D painting/drawing. Components use a mixin pattern (src/gui/components/mixin.ts) for shared functionality and follow A-Frame's component-entity-system architecture. The library integrates with Three.js for 3D geometry creation and Troika for text rendering.

## Key Features

- Dynamic flex-based layout containers with alignment and justification controls
- Interactive GUI components (buttons, checkboxes, sliders, radio buttons) with hover/click states
- Menu systems with dropdown functionality and hierarchical navigation
- 3D text rendering with Troika integration and customizable typography
- Event management hooks for React integration with A-Frame scenes
- Rounded corner geometries and beveled box shapes for modern UI aesthetics
- Canvas-based drawing tools with various brush types (spheres, cubes, lines, stamps)
- Grab-and-drop functionality for VR interactions
- Infinite scrolling lists for large datasets in VR
- Progress indicators and timers with circular visual representations

## Getting Started

Install the package via npm with 'npm install a-frame-components'. Import and call the load() function from the main index to initialize all components: 'import { load } from "a-frame-components"; await load();'. Ensure A-Frame 1.5.0+ is included in your HTML before loading this library. For React integration, use the provided hooks like useEventListenerOn for event handling. Configure TypeScript to include the package's type definitions by adding 'node_modules/a-frame-components/types/**/*' to your tsconfig.json include array. Components can then be used declaratively in A-Frame scenes as custom elements like '<frame-container>', '<frame-button>', etc., or programmatically via the createElement utility function.
