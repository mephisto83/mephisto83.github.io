## Overview

Playwatch-Firebase is a web application that appears to be a streaming video platform or content management system built on Firebase infrastructure. The project combines a React-based frontend with Firebase Cloud Functions for backend services, focusing on user authentication and role-based access control. The application implements a player/master hierarchy system where certain users have elevated privileges to manage content and other users, suggesting it's designed for controlled access to video content or educational materials.

## Architecture

The application follows a Firebase-hosted architecture with three main layers: a React frontend served from the public/ directory containing pre-built static assets, Firebase Cloud Functions in the functions/ directory handling authentication workflows and API endpoints, and Firebase Authentication managing user roles through custom claims. The backend uses Express.js to create REST endpoints (/api/master_me, /api/setCustomClaims) that validate user permissions and assign roles. The system distinguishes between regular players, masters, and an initial master account, with hardcoded user IDs defining the permission hierarchy. The frontend appears to use AdminLTE CSS framework for UI components and includes extensive styling assets.

## Getting Started

1. Install Firebase CLI globally: npm install -g firebase-tools
2. Clone the repository and navigate to the project directory
3. Install root dependencies: npm install
4. Navigate to functions directory and install function dependencies: cd functions && npm install
5. Configure Firebase project: firebase login && firebase use --add (select your Firebase project)
6. Configure firestore.rules and firestore.indexes.json for your Firestore database
7. Update hardcoded user IDs in functions/index.js (PLAYERS, MASTERS arrays) and initial master email
8. Deploy Cloud Functions: npm run deploy
9. Deploy hosting assets: npm run publish
10. Ensure Firebase Authentication is enabled in your Firebase Console with appropriate sign-in methods
11. Configure custom domain and SSL if needed through Firebase Hosting settings
