# PresentationJobs

![Language](https://img.shields.io/badge/language-JavaScript-blue)

## Overview

PresentationJobs is a full-stack web application built as a job marketplace platform that combines ASP.NET MVC/Web API backend with JavaScript frontend components. The application provides a comprehensive job presentation and management system with features for user authentication, job posting, checkout functionality, and real-time communication capabilities. Based on the file structure and dependencies, it appears to be designed as a platform where users can create, manage, and present job opportunities with integrated payment processing and social authentication features.

## Architecture

The application follows a layered ASP.NET MVC architecture with Web API endpoints for data services. The backend is structured with Controllers (AccountController, JobController, CheckoutController, etc.) handling HTTP requests, Identity framework managing authentication and authorization, and Entity Framework for data persistence. The frontend utilizes Knockout.js for MVVM data binding, jQuery for DOM manipulation, and Bootstrap for responsive UI. The application includes a HelpPage area for API documentation, bundle configuration for asset optimization, and OAuth/Bearer token authentication for secure API access. Real-time functionality is implemented through SocketController, and the system supports external authentication via Facebook integration.

## Getting Started

1. Ensure you have .NET Framework and Visual Studio installed. 2. Clone the repository and open Presentation.Jobs.sln in Visual Studio. 3. Restore NuGet packages including Entity Framework, ASP.NET Identity, and OWIN components. 4. Update the connection string in web.config to point to your database server. 5. Run Entity Framework migrations to create the database schema for ApplicationDbContext. 6. Configure Facebook authentication by updating the AppId and AppSecret in Startup.Auth.cs with your Facebook app credentials. 7. Build the solution to ensure all dependencies in bundle.js and CSS files are properly compiled. 8. Configure IIS or use IIS Express to host the application. 9. Run the application and navigate to the home page to verify the setup. 10. Test authentication flows including Facebook login and OAuth token generation.
