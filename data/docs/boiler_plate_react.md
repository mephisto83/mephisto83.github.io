# boiler_plate_react

![Language](https://img.shields.io/badge/language-JavaScript-blue)

## Overview

This repository provides a minimal React boilerplate application with Redux state management, designed as a starting template for React projects. Created in 2016, it implements a basic React-Redux architecture with ES6 support via Babel transpilation and Browserify bundling. The project serves as a foundational scaffold with essential dependencies and build configuration already set up, allowing developers to quickly bootstrap new React applications without configuring the initial toolchain.

## Architecture

The application follows a standard React-Redux architecture pattern with clear separation of concerns. The entry point index.js imports from src/app.js, which configures the Redux store using combineReducers and applies redux-thunk middleware for async actions. The store is created in configureStore() and wrapped around the main App component via React-Redux Provider. Components are organized in src/components/, with the main App component (src/components/app.js) connected to Redux via mapStateToProps and mapDispatchToProps utilities from src/util.js. State management is handled through src/reducers.js which combines individual reducers from src/reducers/reducer.js, while actions are defined in src/actions.js. The build system uses Browserify with Babelify for ES6/JSX transformation.

- ES6/JSX support via Babel transpilation with es2015 and react presets
- Hot reload development workflow using Watchify for automatic bundling
- Pre-configured React-Redux connection utilities in src/util.js
- Modular reducer architecture with combineReducers pattern
- Additional React component libraries included (dropzone, charts, popover, click-outside)
- Simple HTML entry point with bundle.js output
- Browserify-based build system for module bundling

## Getting Started

1. Clone the repository and navigate to the project directory. 2. Install dependencies by running 'npm install' to install all packages listed in package.json. 3. Start the development build process with 'npm run watchify' which will run the Watchify command to monitor changes and automatically rebuild bundle.js. 4. Serve the index.html file using a local server (live-server is included as a dev dependency). 5. Open index.html in a browser to view the application. 6. Begin development by modifying src/components/app.js for UI components, src/actions.js for Redux actions, and src/reducers/reducer.js for state management logic. 7. The build system will automatically transpile ES6/JSX and update bundle.js when files change.
